筆記: 所有客戶都可以使用此功能，但默認情況下未啟用此功能。如果您希望為您的帳戶開啟此功能，請聯絡您的客戶經理。

在“語言設置"中，您可以定義主要語言，然後添加和管理其他元數據語言。主要語言是您的帳戶當前用於視頻元數據的語言。在向您的帳戶添加其他語言之前，現有的元數據將與您定義為主要語言的語言相關聯。在“媒體"模塊中編輯視頻屬性時，最初顯示的是“主要語言"。

設置主要語言

設置主要語言:

  1. 登錄到Video Cloud Studio。
  2. 在導航標題中，單擊行政然後語言設定。請注意，您必須是帳戶管理員才能訪問ADMIN菜單。
  3. 輸入您的主要語言。鍵入時，將顯示匹配語言的列表。
  4. 單擊一種語言以將其選中。
  5. 點擊儲存語言。
要更改主要語言，請單擊該語言，選擇一種新語言，然後單擊保存。

添加其他語言

保存主要語言後，可以添加其他元數據語言。通過添加其他語言，您將能夠在“媒體"模塊中輸入其他多語言元數據。若要添加其他語言，請按照下列步驟。

  1. 點擊新增語言。
  2. 輸入您的語言。鍵入時，將顯示匹配語言的列表。
  3. 單擊一種語言以將其選中。
  4. 點擊儲存語言。該語言將被添加到“其他語言"表中。
  5. 點擊新增語言根據需要添加其他語言。

要編輯語言名稱（它將出現在“媒體"模塊中），請單擊它，進行任何更改，然後單擊“確定"。保存。要刪除語言，請單擊刪除圖標 (）。

筆記: 如果刪除正在使用的語言，則與該語言關聯的所有元數據都將被隱藏。若要還原相關聯的中繼資料，請再次新增語言。
