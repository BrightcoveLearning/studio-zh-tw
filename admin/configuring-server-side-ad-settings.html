廣告配置定義了廣告播放的各個方面，包括廣告調用，計時，信標和其他廣告配置選項。服務器端廣告設置定義可以在視頻點播（VOD）視頻和現場活動期間請求的廣告。服務器端廣告利用Brightcove的服務器端廣告插入（SSAI）技術，並提供以下主要功能:

  • 由於廣告被拼接到服務器端的點播或實時視頻流中，因此它們不會受到廣告攔截器的影響
  • 對於實時事件，廣告會插入到編碼器發送的提示點，或者您可以使用現場模塊控制室
  • 您可以配置填充面板以填充任何未使用的廣告時間

要了解有關服務器端廣告的更多信息，請參閱使用Brightcove Player實施服務器端廣告。

筆記:
  • 在使用SSAI之前，您的視頻雲帳戶需要配置為動態交付並為SSAI啟用。請與您的客戶經理聯繫以獲取詳細信息。
  • 當前不支持Brightcove Live SSAI的VMAP。
  • 儘管SSAI廣告沒有預取功能，但我們將廣告排名邏輯委託給廣告提供商。動態交付使所有添加呼叫都提前進行。如果我們從廣告提供商那裡獲得了一個廣告，並且在我們的系統中對其進行了轉碼，那麼我們會將其插入流中。

“服務器端廣告設置"頁面用於管理實時和VOD廣告配置。請注意，要訪問“服務器端廣告設置"頁面，您必須是具有管理員角色的用戶。

要訪問“服務器端廣告設置"頁面，請執行以下操作:

  1. 登錄到Studio。
  2. 在導航標題中，單擊行政然後服務器端廣告設置。將顯示所有廣告配置的列表。

要創建新的廣告配置，請點擊創建廣告配置然後點擊要創建的特定廣告類型:

  • 視頻點播（VOD）
  • 居住

製作視頻點播廣告配置

筆記: 服務器端廣告配置也可以使用服務器端廣告插入API進行管理。如需詳細資訊，請參閱視訊雲端 SSAI 配置 API 主題。

要添加視頻點播（VOD）廣告配置，請按照以下步驟操作。

  1. 在“服務器端廣告設置"頁上，單擊創建廣告配置>視頻點播。
  2. 進入一個名稱。
  3. 選擇廣告服務器響應:
    • DFP廣告管理系統廣告規則
    • DFP廣告管理系統VMAP
    • 智能XML
    • VAST 3.0
  4. 輸入廣告代碼。
  5. 查看禁用服務器端信標禁用服務器端信標。禁用後，SSAI將不會在服務器端觸發任何信標，並將在VMAP輸出中包括所有信標。
  6. 點擊保存。

要修改廣告配置，請點擊廣告名稱，進行更改，然後點擊保存。要刪除廣告配置，請選中廣告旁邊的框，然後單擊刪除。要復制廣告配置，請選中廣告旁邊的框，然後單擊複製。

創建VOD廣告配置後，打開“播放器"模塊，然後單擊播放器以查看屬性。當。。。的時候啟用服務器端（SSAI）廣告屬性已啟用，可以選擇廣告配置。有關配置播放器廣告的信息，請參閱使用播放器模塊配置播放器廣告。

製作實時廣告配置

筆記:
  • 服務器端實時廣告配置也可以使用Live API進行管理。有關信息，請參閱Brightcove Live API: 服務器端廣告插入（SSAI）。

要添加實時廣告配置，請按照以下步驟操作。

  1. 在“服務器端廣告設置"頁上，單擊創建廣告配置>實時。
  2. 進入一個名稱。
  3. 選擇廣告服務器響應:
    • 廣闊的
    • DFP（廣告規則）
    • 智能XML
  4. 輸入廣告代碼。
  5. 點擊保存。

要修改廣告配置，請點擊廣告名稱，進行更改，然後點擊保存。要刪除廣告配置，請選中廣告旁邊的框，然後單擊刪除。要復制廣告配置，請選中廣告旁邊的框，然後單擊複製。

創建實時廣告配置後，使用啟用服務器端廣告插入（SSAI）屬性啟用後，可以在控制室中選擇廣告配置。有關創建實時事件的信息，請參見使用實時模塊創建和管理實時事件。
