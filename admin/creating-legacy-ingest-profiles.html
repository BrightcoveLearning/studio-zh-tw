提取配置文件管理頁面可用於創建新的舊式提取配置文件。接收配置文件是使用JSON定義的。創建新的舊版配置文件時，查看舊版轉碼的標準攝取配置文件記錄並找到與您需要的配置文件類似的配置文件。然後，在Studio中，您可以輕鬆地複製配置文件，然後使用所做的更改對其進行編輯。

筆記: 有關創建使用DRM的提取配置文件的信息，請參見MPEG-DASH視頻開發人員文檔。

筆記: 每個配置文件最多可以創建25個演繹形式。 海報和縮略圖圖像最多計為25。 

要創建新的舊版提取配置文件，請執行以下操作:

  1. 點擊行政 Studio標頭中的鏈接。
  2. 點擊攝取個人資料關聯。
  3. 找到要復制的舊配置文件。單擊配置文件名稱以打開配置文件編輯器。
  4. 點擊個人資料重複。配置文件編輯器將隨新的配置文件一起打開。
  5. 滾動到配置文件的頂部。編輯name元素，為配置文件賦予有意義的名稱。此名稱將顯示在Video Cloud Studio中。
  6. 編輯description元素，為配置文件提供有意義的描述。
  7. 根據需要編輯移交信息。海報和縮略圖的默認大小位於配置文件的末尾。
  8. 點擊保存。

在配置文件中添加水印

可以通過添加watermarks每個元素rendition: 

       水印”: [ { "y": -20, "x": -10, "width": "20%", "url": "http://www.mysite.com/companylogo.png" } ]

這watermarks元素具有以下參數。

範圍 類型 描述
url 細繩 水印圖片的網址
width 細繩 以像素為單位的寬度或框架寬度的百分比；例如20或10％
height 細繩 以像素為單位的高度或框架寬度的百分比；例如20或10％
x 細繩 從圖像的左邊緣到中心的距離，以像素或幀寬度的百分比表示；例如20％或10％（使用負值（如-10）相對於右邊緣放置水印）
y 細繩 如果圖像是像素或幀高的百分比，則從上邊緣到中心的距離；例如20％或10％（使用負值（如-10）相對於底部邊緣放置水印）
