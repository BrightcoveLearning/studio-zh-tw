當您使用Video Cloud API時，除非您在播放器服務設置中啟用了基本身份驗證，否則方法調用將需要訪問令牌。有關啟用基本身份驗證的更多信息，請參見管理帳戶設置。

筆記: 此功能僅適用於專業和企業帳戶。

創建客戶註冊

“API身份驗證"頁面用於創建新的客戶端註冊。創建註冊時，您可以選擇將在註冊中啟用的服務和角色。

為您的帳戶創建新的客戶註冊:

  1. 點擊行政 Studio標頭中的鏈接。
  2. 點擊API認證關聯。“API身份驗證"頁面將打開，顯示您當前的客戶端註冊。
    筆記: 只有您登入帳號的用戶端註冊才會顯示。
  3. 點擊註冊新申請。
  4. 進入一個名稱和描述用於客戶註冊。
  5. 選擇一個或多個帳戶進行授權。
  6. 選擇要啟用此註冊的API和權限。
    筆記:
  7. 對於 DFP/Google Ad Manager inegration，所需的 API 權限是CMS 視頻閱讀 .
  8. 選擇大量的帳戶和API可能會導致HTTP標頭大小變大且無效。我們的服務器可接受的最大標頭大小為8K，這意味著您必須選擇30個或更少的帳戶來應用憑據。出於安全原因，我們建議您將一組憑據的權限範圍限制為所需的最小帳戶和操作集。
  9. 點擊保存。
  10. 一種客戶編號和客戶機密將被退回。
    筆記: 複製用戶端密碼並將其儲存在安全的位置，因為用戶端密碼將不會再次顯示。

管理客戶註冊

“API身份驗證"頁面可用於查看和管理客戶端註冊。點擊行政 Studio標頭中的鏈接，然後單擊API認證。將顯示您當前註冊的列表。

要查看註冊詳細信息，請單擊名稱。在這裡，您可以:

  • 修改名字
  • 複製客戶編號
  • 添加授權帳戶
  • 選擇其他API以啟用
  • 刪除憑證

點擊保存保存您的更改。

請注意，如果為多個帳戶創建了憑證，則用戶只有在有權訪問所有這些帳戶的情況下才會看到該憑證。例如，用戶1為帳戶A和B創建憑據。用戶2登錄並僅具有對帳戶A的訪問權限。用戶2將看不到該憑據，因為他們無權訪問帳戶B。

刪除客戶註冊

要刪除客戶註冊，請單擊刪除圖標（) 旁邊的註冊或編輯註冊並單擊刪除註冊 .刪除註冊將永久禁用它，並且無法將其撤消。

使用DFP廣告管理系統

如果您當前在DFP廣告管理系統中將多個Video Cloud帳戶設置為內容源，則有兩個選擇:

  1. 您可以為在DFP廣告管理系統中設置為內容源的每個視頻雲帳戶創建唯一的客戶端ID和客戶端密鑰。例如，如果您有10個Video Cloud帳戶，則Video Cloud中的API身份驗證設置中將有10個名為DFP的應用程序。
  2. 您可以為已在DFP中設置為內容源的所有Video Cloud帳戶創建單個Client ID和Client Secret。例如，如果您有10個Video Cloud帳戶，則Video Cloud中的API身份驗證設置中將有1個名為DFP的應用程序。

無論您選擇選項＃1還是＃2，對於在DFP廣告管理系統中設置的每個內容源，都需要提供一個唯一的帳戶ID。因此，您不會減少DFP廣告管理系統中當前設置的內容源的數量；您只是將讀取令牌替換為客戶端ID和客戶端密鑰，這些令牌在內容源之間可以相同，也可以在每個內容源之間唯一。

選項＃1需要花費更多時間。但是，如果所有內容源中的Client ID和Client Secret都相同（與選項＃2一樣），則添加新的內容源將是一個負擔，因為您將需要生成新的Client ID和Client Secret並進行更新DFP廣告管理系統中的所有內容源。因此，Brightcove建議使用選項1。
