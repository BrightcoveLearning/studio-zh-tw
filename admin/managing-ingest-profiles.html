提取配置文件包含使用以下格式上傳視頻時將使用的轉碼設置: 上載模塊或者動態提取API。

要訪問“摘要配置文件"管理頁面，請點擊行政鏈接然後攝取個人資料。請注意，要訪問“接收配置文件"管理頁面，您必須是具有管理員角色的Video Cloud Studio用戶。

將會出現一個攝取配置文件列表。

筆記: 點擊顯示API名稱標頭中的鏈接將顯示攝取配置文件的內部名稱。在 API 呼叫中使用內嵌設定檔名稱時，會使用內部名稱。

該頁面將顯示以下信息:

  • 配置文件名稱和描述
  • 個人資料版本 -指示使用的轉碼類型。請注意，顯示的類型取決於您的視頻雲帳戶的配置方式:
    • 動態投放 -使用動態傳遞轉碼。有關動態投放的信息，請參見動態交付概述。
    • 情境感知  -使用上下文感知轉碼。有關上下文感知編碼的信息，請參見上下文感知編碼概述。
  • 類型 -攝取檔案的類型
    • 標準視頻點播 -表示這是Video Cloud提供的默認配置文件。此類個人資料無法編輯或刪除。
    • 自定義視頻點播 -表示這是一個自定義攝取配置文件。
    • 標准直播 -表示這是Video Cloud提供的默認實時配置文件。此類個人資料無法編輯或刪除。
    • 自訂直播 -表示這是一個自定義實時提取配置文件。
  • 演繹 -可以創建的視頻解譯數量。
  • 展示 -確定是否顯示個人資料並可以在“上傳"模塊中選擇

有關Brightcove提供的攝取配置文件（帶有類型的標準視頻點播或者標准直播），請參閱以下文件之一:

  • 動態交付或上下文感知配置文件版本 -- 用於動態交付和上下文感知編碼的標準攝取配置文件
  • 自定義直播類型 -- 標準實時攝取配置文件

使用“接收配置文件"管理頁面，您可以:

  • 創建新的攝取配置文件
  • 編輯攝取配置文件
  • 刪除攝取配置文件
  • 更改默認的攝取配置文件
  • 隱藏/顯示攝取配置文件

創建一個新的攝取配置文件

筆記: 建立和編輯內嵌設定檔的功能僅適用於具有企業和專業帳戶的發行者。

要創建新的提取配置文件，請點擊新的配置文件。

創建新的攝取配置文件的過程將根據所創建的配置文件的類型而有所不同。有關如何創建自定義提取配置文件的詳細信息，請參閱以下文檔之一:

  • 創建用於動態交付的提取配置文件
  • 創建用於上下文感知編碼的攝取配置文件
  • 創建自定義實時攝取配置文件

編輯攝取配置文件

攝取配置文件管理頁面可用於編輯攝取配置文件。請注意，默認的攝取配置文件集（已標記為標準視頻點播）無法編輯。

要編輯攝取配置文件，請執行以下操作:

  1. 點擊行政 Studio標頭中的鏈接。
  2. 點擊攝取個人資料關聯。
  3. 單擊攝取配置文件進行編輯。
  4. 使用概要文件編輯器來編輯概要文件定義。
  5. 點擊保存。

刪除攝取配置文件

“攝取配置文件"管理頁面可用於刪除不再需要的攝取配置文件。請注意，默認的攝取配置文件集（已標記為標準視頻點播）無法刪除，但他們可以被隱藏如果不需要它們。

要刪除提取的個人資料，請執行以下操作:

  1. 點擊行政 Studio標頭中的鏈接。
  2. 點擊攝取個人資料關聯。提取配置文件管理頁面將打開，顯示為您的帳戶創建的提取配置文件。
  3. 單擊刪除圖標 () 旁邊的攝取配置文件或單擊配置文件名稱，然後單擊刪除個人資料 .
  4. 通過單擊確認刪除刪除個人資料。

選擇默認的攝取配置文件

可以針對以下情況配置默認的攝取配置文件: 視頻已上傳什麼時候使用社交模塊裁剪。上傳默認配置文件用 在配置文件名稱旁邊。社交剪輯默認配置文件用 在配置文件名稱旁邊。

要選擇默認的上傳配置文件進行上傳/剪切，請執行以下操作:

  1. 點擊行政 Studio標頭中的鏈接。
  2. 點擊攝取個人資料關聯。將打開“攝取配置文件"管理頁面，顯示為該帳戶創建的攝取配置文件。
  3. 單擊您想要設置為默認配置文件的攝取配置文件名稱。
  4. 點擊上載中或者社會剪裁頁面頂部的鏈接將當前配置文件設置為默認配置文件。
筆記: 選取預設的內嵌設定檔時，您應該確定設定檔是可見且不會隱藏的。
筆記: 只能將“動態投放"配置文件設置為社交剪輯默認配置文件。不支援 CAE 設定檔。

隱藏和顯示攝取配置文件

通過將與配置文件相關聯的滑塊單擊到隱藏位置。這對於隱藏可能不需要的默認配置文件很有用。

隱藏的攝取配置文件不會出現在“上載"模塊的“攝取配置文件"列表中。
