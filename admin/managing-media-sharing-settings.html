媒體共享使發布者可以一次上傳視頻，然後將其共享到其他帳戶。這些帳戶可以是您自己的公司帳戶，也可以是其他發布者的帳戶。然後可以在其他視頻雲帳戶中獨立使用共享的視頻。媒體共享設置用於管理帳戶之間的共享關係。

筆記: 媒體共享僅適用於Video Cloud Enterprise客戶。如果您有興趣升級您的視頻雲帳戶，請聯繫 Brightcove 以獲取更多信息。

媒體共享也可以使用CMS API以編程方式實現。有關更多信息，請參見使用CMS API進行媒體共享。

本主題將使用以下術語:

  • 掌握 -創建原始視頻的帳戶。主人擁有內容，並負責設置，管理和向會員提供內容。
  • 會員關係 -正在接收視頻的帳戶。關聯企業可以接受來自母版的共享內容。

啟用媒體共享

要啟用媒體共享，請聯繫Brightcove客戶支持並讓他們為您的帳戶打開媒體共享。

添加會員

要將視頻共享到另一個Video Cloud帳戶，必須添加會員帳戶。會員是您要共享視頻的帳戶。若要添加會員帳戶，請按照下列步驟操作:

  1. 以管理員身份登錄Video Cloud Studio。
  2. 在導航標題中，單擊行政然後媒體分享。請注意，您必須是帳戶管理員才能訪問ADMIN菜單。
  3. 這管理會員-與我共享視頻的帳戶部分將顯示與您建立媒體共享關係的所有會員發布者的列表。
  4. 要添加新的會員，請在帳戶ID文本框，然後單擊添加會員。

該帳戶將被添加到表中，並帶有關係狀態的待辦的。會員帳戶將收到媒體共享通知，並能夠接受或拒絕關係。如果會員接受媒體共享關係，則關係狀態將公認。

自定義字段匹配

要要求會員帳戶具有與主帳戶相同的自定義字段，請選中實施自定義字段匹配然後點擊更新規則。什麼時候實施自定義字段匹配如果選中，則附屬帳戶必須具有與主帳戶相同的自定義字段，否則任何共享都會失敗。

筆記

  • 自定義字段匹配只會驗證兩個帳戶中的內部自定義字段名稱是否匹配。字段的類型（列表或文本）無關緊要。

  • 如果主帳戶定義的頻道強制執行自定義字段匹配，然後共享具有自定義字段且其關聯會員帳戶不允許的值的視頻，則共享嘗試將失敗。

    如果主帳戶定義的頻道沒有強制執行自定義字段匹配，然後共享具有自定義字段且其關聯會員帳戶不允許的值的視頻，則共享嘗試將起作用，但存在不良情況的字段值不會包含在視頻的會員副本中。

  • 筆記: 如果主帳戶中的共享視頻配置了地理過濾，則還必須啟用會員帳戶進行地理過濾，否則共享將失敗。在共享視頻上配置的所有地理位置過濾也將應用於會員帳戶。要啟用一個帳戶進行地理位置過濾，請聯繫Brightcove支持。

刪除會員

要刪除附屬帳戶，請找到要刪除的帳戶並單擊刪除圖標 (）。將出現一個對話框，確認您要終止共享關係。

點擊結束分享結束共享關係。

批准/拒絕媒體共享關係

如果另一個視頻雲已將您的帳戶添加為會員（想與您的視頻雲帳戶共享視頻），則當管理>媒體共享頁面被訪問。

如果實施自定義字段匹配如果選中，則附屬帳戶必須具有與主帳戶相同的自定義字段，否則任何共享都會失敗。

點擊拒絕拒絕媒體共享關係。否則，選擇一個接受方式:

  • 自動的 -始終接受來自該主帳戶的共享視頻。共享的視頻將出現在視頻的媒體模塊列表中。
  • 手動的 -將主視頻中的共享視頻添加到媒體庫之前，請先對其進行審核。媒體模塊將顯示一個股票待定左側導航欄中的項目。

點擊批准批准媒體共享關係。關係獲得批准後，它將顯示在管理大師頁面的部分。

點擊查看合同查看媒體共享設置。這接受方式滑塊可用於更改接受方式。要刪除共享關係，請單擊刪除圖標 (）。
