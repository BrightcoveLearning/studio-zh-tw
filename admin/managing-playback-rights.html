播放權限用於限制視頻播放。創建播放權限後，可以使用“媒體"模塊將它們分配給視頻。

要訪問“播放權限"管理頁面，請點擊行政鏈接然後播放權。將出現播放權限列表。

筆記: 若要存取 [播放權限] 頁面，您必須是具有管理員角色的 Video Cloud Studio 使用者。
播放權分配給了播放限制媒體模塊中視頻屬性頁的“視頻"部分。

播放權地位的不活躍無法分配給“媒體"模塊中的視頻。

創建播放權限

要創建播放權限，請按照下列步驟操作。

  1. 以管理員身份登錄Video Cloud Studio。
  2. 在導航標題中，單擊行政然後播放權。請注意，您必須是帳戶管理員才能訪問ADMIN菜單。
  3. 點擊新播放權。
  4. 進入一個播放權限名稱。
  5. 檢查允許沒有令牌的播放權將允許在沒有JSON Web令牌（JWT）的情況下發出播放請求。
    筆記: 取消選中此選項不會影響為該“播放權限"設置的限制，但是，播放請求將需要一個有效的JWT，該JWT是在播放器級別以編程方式設置的。如需詳細資訊，請參閱使用播放授權 DRM 與 Brightcove 播放程式的開發人員文件。
  6. 查看默認使用將其設為默認播放權限。這會將配置的播放權限限制應用於上載的視頻。
    筆記: 如果視訊重新轉碼，則不會指派預設的播放權限。
  7. 點擊保存。
筆記: 帳戶中最多可以有100個播放權限。如果您需要增加上限，請聯絡布萊特灣支援部門。

單擊左側導航中的限制鏈接，以向播放權限添加限制:

  • 地域
  • 領域
  • 代理人
  • 時間

編輯播放權限

若要編輯播放權限，請按照下列步驟操作:

  1. 單擊播放權。
  2. 進行必要的更改，然後單擊保存。

複製播放權限

若要復製播放權限，請按照下列步驟操作:

  1. 單擊播放權。
  2. 點擊複製。
  3. 進行必要的更改，然後單擊保存。

激活/停用播放權限

播放權限無法刪除，但是可以將其禁用，因此不能將其分配給“媒體"模塊中的視頻。若要激活/停用播放權限，請按照下列步驟操作:

  1. 單擊播放權。
  2. 點擊啟用激活播放權或停用取消播放權。無法將已停用的播放權限分配給“媒體"模塊中的視頻。

分配地理限制

地理限制允許您根據國家、郵政編碼、IP 和指定的市場區域限制視頻播放。要將地理限制分配給播放權限，請按照下列步驟操作:

  1. 單擊播放權。
  2. 在左側導航中，單擊地理限制。

這全局規則可以設置為允許所有或者全部屏蔽 .

  • 允許所有
    • 當您想將少數無權訪問的組列入黑名單時使用
    • 選擇允許所有人查看，然後添加禁止查看的黑名單國家、郵政編碼、IP 和 DMA 列表
  • 全部屏蔽
    • 當您想將少數有權訪問的組列入白名單時使用
    • 選擇阻止所有人查看，然後添加允許查看的白名單國家、郵政編碼、IP 或 DMA 列表

可以添加地理限制以允許/阻止查看基於:

  • 國別
  • 郵遞區號
  • IP限制
  • 指定市場區域（DMA）

限制進入國家

國家/地區播放權限可用於限制特定國家/地區的觀看。

  1. 選擇將被阻止（列入黑名單）或允許（列入白名單）查看的國家/地區。鍵入時，將過濾選擇列表。
  2. 點擊保存。

限制對郵政編碼的訪問

郵政編碼播放權限可用於限制查看特定郵政編碼。

  1. 輸入將被阻止（列入黑名單）或允許（列入白名單）查看的郵政編碼列表。郵政編碼前應加上大寫字母，兩個字母的國家/地區代碼和連字符，例如， US-90045。

還可以上傳包含郵政編碼的 .csv 文件。點擊上載CSV瀏覽並選擇要上傳的文件。

筆記: 上傳 .csv 檔案將取代郵遞區號欄位的內容。

限制對 IP 的訪問

IP 播放權限可用於將查看限制在 IP 地址範圍內。

  1. 輸入將被阻止（列入黑名單）或允許（列入白名單）查看的 IP 地址範圍。CIDR 塊符號也支持。
  2. 點擊保存。

限制對 DMA 的訪問

指定市場區域 (DMA)，也稱為媒體市場，是美國的一個區域，用於定義電視和廣播市場。播放權限可用於限制基於 DMA 的觀看。

  1. 輸入將被阻止（列入黑名單）或允許（列入白名單）查看的 DMA 列表
  2. 點擊保存。

分配域限制

域限制允許您根據域限制視頻播放。要將域限制分配給播放權限，請按照下列步驟操作:

  1. 單擊播放權。
  2. 在左側導航中，單擊域限制。
  3. 在裡面列入黑名單的域字段中，輸入將無法查看的域。
  4. 在裡面列入白名單的網域字段中，輸入將允許查看的域。
  5. 點擊保存。

分配代理限制

代理限制使您可以根據某些類型的連接來限制視頻播放。要將代理限制分配給播放權限，請按照下列步驟操作:

筆記: 隨著 VPN 提供商不斷改進他們的服務，解析 IP 地址可能會與您為阻止的 IP 地址定義的不同。正因為如此，代理阻塞並不是 100％ 的保證。
  1. 單擊播放權。
  2. 在左側導航中，單擊代理限制。
  3. 選擇要阻止的代理類型。
  4. 點擊保存。

分配時間限制

時間限制允許您根據指定的開始日期限制視頻播放。要為播放權限分配時間限制，請執行以下步驟:

  1. 單擊播放權。
  2. 在左側導航中，單擊時間限制 .
  3. 單擊日曆控件以選擇一個開始日期 .
  4. （可選）點擊添加結束日期選擇結束日期。
  5. 點擊保存。
