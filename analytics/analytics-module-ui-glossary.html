筆記: 如需 Analytics API 傳回的資料項目清單，以及匯出分析資料時，請參閱分析 API 詞彙表。

一種

主動媒體
帳戶中的活動視頻數。請注意，這些數據不是實時的-每天從您的視頻庫同步一次。另請注意，如果您正在查看“實時"分析數據（少於3天），則此數據可能不准確。
分析 API
基於REST的應用程序編程接口，使開發人員可以直接以JSON格式檢索視頻雲視頻的分析指標。這分析 API為 Video Cloud Studio 中的 Analytics UI 提供數據。

乙

總帶寬
用於交付視頻，圖像和播放器負載的總帶寬。請注意，帶寬數據的某些部分會延遲，因為它們來自CDN等外部來源。通常情況下，信息會在48小時內完成，但是在某些情況下，一天可能長達5天。
瀏覽器類型
瀏覽器類型。典型值為:
  • 鉻合金
  • 蘋果瀏覽器
  • 火狐
  • IE
已傳送位元組
發送給觀看者的總數據（以字節為單位），包括視頻/音頻數據（包括緩衝數據（無論是否播放），海報和縮略圖以及與視頻播放器相關的任何數據）。

C

城市
觀看視頻的城市。
國家
觀看視頻的國家/地區的英文簡稱。遵循ISO 3166標準。
CSV
逗號分隔值。可以以這種格式導出Analytics數據。

d

每日唯一身份用戶
這是視頻在一天中擁有的唯一身份觀看者的估計數量。該數據僅是可用的對帳數據（3天后），並且僅在帳戶級別。有關更多信息，請參見Analytics API: 不重複使用者。
日
觀看視頻的當月日期。
設備
的組合設備類型和設備操作系統用於觀看視頻。
設備操作系統
設備上用於觀看視頻的設備操作系統。可能的值為:
  • 安卓
  • 巴達
  • 的iOS
  • 的Linux
  • OS X
  • RIM（黑莓）
  • 塞班
  • 網絡操作系統
  • 視窗
  • 其他-未映射到以上任何一個的用戶代理
設備製造商
設備的製造商。一些典型值是:
  • 聯想
  • 蘋果
  • 索尼
  • 微軟
設備類型
用於觀看視頻的設備類型。可能的值為:
  • 台式機-包括台式機和筆記本電腦
  • 平板電腦-包括iPad，Android平板電腦等。
  • 手機-手機
  • 連接的電視/遊戲機/伴侶設備-包括智能電視，機頂盒和遊戲機
  • 其他-未映射到以上任何一個的用戶代理
 
尺寸
維度是可以報告和過濾的分析數據類別。當前可用的尺寸為:
  • 播放器
  • 視頻
  • 國家
  • 日
  • 顯示域
  • 設備
  • 設備類型
  • 設備操作系統
  • 流量來源
 
顯示域
客戶端（瀏覽器或其他）報告的引薦域。
顯示網址
觀眾觀看視頻時使用的URL。

參與度得分
每個視頻觀看的平均觀看百分比（通過將所有觀看百分比與視頻觀看次數相加得出）。有關如何計算的更多信息，請參見計算視頻參與度得分。

流量來源
觀眾如何到達播放器頁面的指示。可能的值為:
  • 引薦-用戶點擊了該視頻的鏈接
  • 直接-用戶直接輸入鏈接，使用書籤或使用電子郵件中的鏈接
  • 自然搜索-用戶執行搜索，然後點擊搜索結果以觀看視頻
  • 付費搜索-用戶執行了搜索，然後點擊了付費結果以觀看視頻
  • 安全搜索-用戶使用Google帳戶登錄，執行搜索並點擊了搜索結果（付費或自然搜索；這種情況下不包括搜索信息）
