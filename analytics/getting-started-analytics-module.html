視頻雲分析平台是由Brightcove使用大數據技術的最新創新從頭開始構建的，最先進的，接近實時的分析平台，並進行了大規模構建。該分析平台提供對分析數據的快速，近乎實時的訪問，並包括一個強大的開放API，以使開發人員可以創建自定義分析解決方案和報告，以及Video Cloud Studio中所包含的那些報告。

 

無論您對在線視頻有何具體目標，了解受眾都是製定成功視頻策略的關鍵。您對觀眾的了解越多，創建的內容就越容易吸引他們，使其易於使用。Google Analytics（分析）模塊提供了不同的報告，可用於獲取有關您的視頻和播放器的分析:

  • 績效報告 -提供有關視頻觀看次數，流量來源，設備類型和設備操作系統的詳細信息
  • 參與度報告 -提供有關用戶如何與您的視頻互動的詳細信息；觀看視頻的觀眾所佔的百分比
  • 現在趨勢 -提供過去一小時的並發觀眾數
  • 風俗 -提供創建自定義分析報告的功能

使用這些報告，您可以生成有關擁有多少觀眾，觀眾正在觀看多少內容，何時停止觀看視頻甚至內容的流量來源的最新報告。

本主題包括以下各節，介紹如何使用Google Analytics（分析）模塊:

  • 啟動分析模塊
  • 選擇時間跨度
  • 搜索視頻
  • 局限性

請注意，Analytics模塊中顯示的所有數據也可以通過Analytics API獲得。有關更多信息，請參見視頻雲分析API文檔。

啟動分析模塊

要啟動Google Analytics（分析）模塊，請執行以下操作:

  1. 登錄到Video Cloud Studio。
  2. 通過單擊打開分析模塊分析工具在導航標題中。

選擇時間跨度

使用日期選擇器指定報告的時間段，您可以選擇固定日期範圍或選擇自定義日期範圍。請注意，選擇一天后，圖表將按小時顯示。

報告時區

使用帳戶的報告時區顯示“分析"模塊中的數據。時區顯示在日曆控件下方。

報告時區用於計算報告數據的日期範圍。更改報告時區設置只會影響以後的數據，並且不會追溯應用更改。如果更改此設置，則在查看應用更改的日期範圍時，數據中可能會出現平坦點或峰值。對該設置的更新可能不會立即生效，並且分析數據可能會繼續引用舊設置，直到系統處理更改為止。

報告時區可以配置為帳號設定。有關更改報告時區的信息，請參見管理帳戶設置。

搜索視頻

搜索功能可用於搜索特定視頻。

將搜索以下視頻屬性:

  • 名稱
  • 簡短的介紹
  • 詳細描述
  • 標籤
  • 影片編號
  • 參考編號
  • 所有自定義字段

匹配列表將顯示在搜索字段下方。

單擊視頻鏈接將打開“上下文"報告。例如，如果您正在查看帳戶的參與度，然後使用搜索找到特定的視頻，則單擊該視頻的鏈接將打開“參與度報告"。如果您正在查看效果報告，則搜索並單擊鏈接將打開效果報告。

有關在Google Analytics（分析）模塊中搜索視頻時可用選項的完整列表，請參見如何在Google Analytics（分析）模塊中搜索視頻。

筆記: Google Analytics（分析）報告的確包含對已從您的帳戶中刪除的視頻的分析。您無法在Google Analytics（分析）模塊中搜索已刪除的視頻。若要取得已刪除影片的報表，您必須知道已刪除影片的影片 ID，並使用 Analytics API 來產生報表。
