使用快速查找

“分析"模塊中的“快速查找"功能可用於搜索特定視頻。輸入搜索條件後，快速查找將搜索視頻名稱，簡短描述，詳細描述，標籤，視頻ID和您可能擁有的任何自定義文本字段。匹配列表將顯示在搜索字段下方。

注意已刪除視頻無法通過搜索功能進行搜索，但我們會為它們保留分析。您可以藉由展開/瀏覽效能報告，並使用瀏覽器搜尋功能，或是下載報表並在試算表應用程式中進行搜尋，即可找到這些項目。

單擊匹配項的鏈接將打開“在上下文中"的報告。例如，如果您正在查看帳戶的參與度，然後使用“快速查找"找到特定的視頻，則單擊該視頻的鏈接將打開“參與度報告"。如果您正在查看效果報告，則搜索並單擊鏈接將打開效果報告。

基本搜尋

如果您提供的搜索值沒有限定符，則將對以下字段執行文本搜索:

  • 名稱
  • 簡短的介紹
  • 詳細描述
  • 標籤
  • 影片編號
  • 參考編號
  • 所有自定義字段

如果要匹配多個單詞，只需將其用引號引起來。

高級搜索

Google Analytics（分析）模塊還提供了以下功能搜索特定字段，根據日期搜索和包含或排除特定術語。

搜索特定字段

要在特定字段中搜索指定的值，請使用以下語法:

  • field:value to search

支持以下字段:

場地 支持的值
name 字符串或帶引號的字符串
text 字符串或帶引號的字符串（值name，short description，long description）
tags 字符串或帶引號的字符串
reference_id 字符串或帶引號的字符串
state ACTIVE，INACTIVE，DELETED，PENDING
updated_at 看搜索特定日期日期格式
created_at 看搜索特定日期日期格式
published_at 看搜索特定日期日期格式
complete true或者false

如果您想進行完全匹配或匹配多個單詞，只需將其用引號引起來即可:

  • "live"
  • "live streaming"

當針對特定字段進行搜索時，這也適用:

  • name:"live"
  • name:"live streaming"

搜索特定日期

與日期相關的搜索字段採用日期範圍值。支持多種用於指定日期的格式:

格式 意義
2013-08-01 yyyy-mm-dd格式的日期
2013-08-01T00: 00: 00Z UTC格式的時間
-1天 當前時間減去1天（請參見相對日期）
+1d 當前時間加上1天（請參閱相對日期）
 

相對日期

對於相對日期，我們支持一個方向（+或-），後跟一個數字，然後是一個持續時間（或其縮寫）。合法持續時間為: 分鐘（m），小時（h），天（d）

例如:

  • -1day
  • -2days
  • +4hours
  • -60minutes
  • -1d
  • -2d
  • +4h
  • -60m

日期範圍

如果要搜索的字段是日期，則可以指定要搜索的日期範圍:

  • updated_at:2013-08-01T00:00:00Z..2013-10-08T00:00:00Z

這將搜索所有帶有updated_at在2013年8月1日至2013年10月8日之間的值。

開放式範圍

如果要匹配直到給定時間的所有日期，或者要匹配到給定時間的所有日期，請不要使用範圍的一端。例如，要搜索最近2天修改過的所有視頻，請執行以下操作:

  • q=updated_at:-2days..

搜索2013年8月11日之前修改的所有視頻:

  • q=updated_at:..2013-08-11T00:00:00Z

 

包括和排除特定術語

您可以根據需要標記一個字詞（返回的視頻必須匹配）或禁止（返回的視頻必須不是比賽）。這是由一個+或者-緊接該術語之前簽名:

例子 意義
+animals 視頻必須包括術語 '' 的動物name , short description ,或long description
-animals 影片' 中不得包含「動物」一詞name，short description，或long description
+name:animals 視頻必須包括術語 '動物' 的名稱
-name:animals 影片不得在名稱中包含「動' 物」一詞
