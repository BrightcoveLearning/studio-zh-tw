使用平視顯示器Chrome擴展程序

{{ page.description }}

平視顯示器（HUD）是Chrome擴展程序，它可以執行以下操作:

  • 查看實時頁面上視頻的性能指標
  • 發布，編輯和刪除您帳戶中的視頻

安裝Chrome擴展程序

要安裝抬頭顯示器Chrome擴展程序，請按照以下步驟操作:

  1. 使用Google Chrome網絡瀏覽器，訪問Brightcove Campaign擴展頁面這裡。
  2. 點擊新增至Chrome。
  3. 出現提示時，單擊添加擴展名。
  4. 通過在地址欄旁邊找到Brightcove徽標，確認擴展程序已安裝。

打開平視顯示器

平視顯示器對於查看視頻分析以及發布，編輯和刪除帳戶中的視頻非常有用。要打開HUD，請按照下列步驟操作:

  1. 打開谷歌瀏覽器。
  2. 單擊地址欄旁邊的Brightcove徽標以打開擴展名。
  3. 如果您與Brightcove Campaign應用程序之間沒有活動的會話，系統將提示您登錄。輸入您的電子郵件地址和密碼。點擊登入。

HUD頂部的圖標可用於:

  • 審核分析
  • 管理影片

點擊上網將在瀏覽器窗口中打開Brightcove Campaign應用程序。

選擇一個賬戶

HUD的右上方將顯示當前的Campaign帳戶。

如果您有權訪問多個Campaign帳戶，則單擊帳戶名稱將打開帳戶選擇器。點擊帳戶名稱下拉菜單中選擇要使用的其他帳戶。

要退出HUD，請單擊登出。

審查視頻分析

首次打開HUD時，它將顯示視頻分析。您還可以單擊分析圖標 () 進入分析模式。HUD將僅在使用Brightcove Campaign™發布的頁面上顯示視頻分析。

HUD將在頁面上顯示有關視頻的基本信息:

  • 影片名稱
  • 影片編號
  • 影片內容類型
  • 創建日期

如果頁面上有多個視頻，上一個視頻和下一個視頻按鈕將被啟用。

HUD上的視頻分析“接近實時"。分析通常會延遲幾分鐘，但是在流量高峰時，延遲會增加。將顯示過去30天的以下分析數據:

  • 影片曝光 -加載視頻並準備播放的次數
  • 視頻開始 -視頻開始播放的次數
  • 平均觀看時間 -總觀看時間/視頻開始播放
  • 視頻基準得分 -總體評分，用於衡量您的視頻的效果（學到更多...）
  • 播放率 -視頻開始/視頻展示
  • 平均觀看百分比 -平均觀看時間/視頻時長

要查看視頻的所有分析，請點擊查看所有分析關聯。

管理影片

要使用 HUD 管理和發布您帳戶中的視頻，請單擊視頻圖標 ()。帳戶中的視頻列表將出現。

將鼠標懸停在視頻上將顯示一系列按鈕，這些按鈕可以:

  • 發布視頻
  • 編輯影片
  • 刪除影片

發布影片

將鼠標懸停在視頻上，然後點擊發布。選擇一個目的地，網頁或者電子郵件。該嚮導將引導您完成生成嵌入代碼或獲取視頻縮略圖以在電子郵件中使用的過程。

編輯影片

將鼠標懸停在視頻上，然後點擊編輯。根據需要編輯視頻屬性，然後單擊保存。

刪除影片

將鼠標懸停在視頻上，然後點擊刪除。系統將提示您確認刪除。
