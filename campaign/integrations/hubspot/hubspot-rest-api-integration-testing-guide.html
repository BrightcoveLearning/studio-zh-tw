當使用REST API設置新的HubSpot集成時，測試並確認視頻觀看數據確實已交付給HubSpot可能是具有挑戰性的。在本主題中，您將學習如何系統地測試和確認集成是否有效以及如何將查看數據和銷售線索發送到HubSpot。

筆記: 本主題適用於驗證使用 REST API 所建立的中樞連線。
筆記: 如需有關如何設定廣告活動模組以與 HubSpot 搭配使用的資訊，請參閱使用 REST API  整合視訊雲端與 HubSpot 。

開始之前

在驗證集成之前，假定您具有:

  • 成功完成了Brightcove Campaign 與 HubSpot 的集成
  • 至少創建了一個支持戰役的播放器並將其配置為使用 HubSpot 連接

配置測試線索表格並發布視頻

出於測試目的，您需要在 Campaign 中創建和配置自定義潛在客戶表單。這是一個 HubSpot 表單，您已將其嵌入代碼粘貼到 Campaign 中。即使您不會在生產環境中使用潛在客戶表單，也可以使用該潛在客戶表單來確保您在數據庫和頁面中是已知的。要創建 HubSpot 表單並配置 Campaign 以使用它，請執行以下步驟。

  1. 登錄到您的HubSpot帳戶。
  2. 創建並設置HubSpot表單的樣式。有關創建HubSpot表單的信息，請參見為HubSpot創建自定義潛在客戶表單。
  3. 點擊分享在頁面頂部。
  4. 將表單嵌入代碼複製到剪貼板。
  5. 返回活動模塊。
  6. 使用HubSpot表單嵌入代碼創建新的潛在顧客表單。
  7. 設置您的自定義窗體以在5秒標記處顯示該窗體。這樣，您將確切知道何時應顯示它。
  8. 選中復選框即使已經知道潛在客戶，也要始終顯示潛在客戶表格。這樣可以確保在測試過程中填寫完表格後繼續顯示該表格。
  9. 保存表格。
  10. 配置具有HubSpot連接的Brightcove Player，以使用剛剛創建的HubSpot潛在客戶表單。

生成預覽鏈接以觀看視頻

要生成視頻的預覽鏈接，請按照下列步驟操作。

  1. 打開媒體模塊。
  2. 選擇一個視頻並將其發佈到網絡播放器。確保在潛在客戶表單中選擇啟用了戰役的玩家。
  3. 單擊預覽鏈接以在新的瀏覽器窗口中打開視頻。通過這種方法進行測試，可以消除HTML頁面上嵌入的播放器可能引入的任何潛在變量。
  4. 確認視頻已播放，並且線索表顯示在5秒標記處。
  5. 使用唯一的聯繫信息和電子郵件地址（即名人名或包含“bctest"的名字）填寫潛在客戶表單，以便以後可以在HubSpot中輕鬆找到它。這消除了任何瀏覽器，播放器或數據庫級別的變量。通過以新用戶身份填寫表單，我們可以確保您現在是HubSpot中的已知用戶。

確認 Campaign 成功捕獲數據

在 HubSpot 中查找視頻觀看數據之前，首先確認 Campaign 確實捕獲了視頻觀看。請注意，查看數據最多可能需要 3 小時才能在營銷活動 UI 中顯示。要確認 Campaign 成功捕獲數據，請執行以下步驟。

  1. 打開活動模塊。
  2. 點擊最近所有活動在左側導航欄中。
  3. 確認有一個帶有最近時間戳記的事件，該事件顯示了剛剛觀看的視頻的名稱。事件狀態應為未處理。這表示 Campaign 已捕獲視圖但尚未嘗試將其同步到 HubSpot。
  4. 要手動觸發 Campaign 和 HubSpot 之間的同步，請單擊同步活動在左側導航中。
  5. 來自選擇連接下拉列表中，選擇您的HubSpot集成。一種測試同步按鈕應該出現。
  6. 點擊測試同步觸發手動同步。請注意，您限於觸發手動同步的頻率。默認情況下，Campaign 每小時將視圖同步到 HubSpot 一次。
  7. 點擊最近所有活動在左側導航欄中。
  8. 您的觀看事件狀態為排隊。數據實際同步到HubSpot可能需要15分鐘左右。數據同步後，狀態將更改為以下兩種狀態之一:
    • 已同步 -表示該用戶是已知用戶，並且數據已成功傳送到其活動日誌。幾分鐘後，數據也應該在HubSpot中可見。
    • unknown_contact -表示該用戶被視為未知用戶，並且HubSpot無法接受數據。如果你得到一個unknown_contact狀態，請檢查您是否使用正確的播放器發布了視頻，是否已安裝HubSpot表單以及是否正在使用預覽鏈接（而不是嵌入在頁面中）進行測試。

確認視頻觀看數據在HubSpot中

一旦廣告系列顯示已同步狀態，請按照以下步驟驗證數據是否在 HubSpot 中。

筆記: HubSpot UI 需要 5-10 分鐘才能使用從廣告活動收到的新資料進行更新。
  1. 登錄到您的HubSpot帳戶。
  2. 在HubSpot中，導航到聯絡人區域並找到您填寫潛在客戶表單時使用的唯一名稱或電子郵件地址。
  3. 單擊聯繫人以打開活動日誌。
  4. 點擊過濾活動下拉菜單（頁面頂部的活動菜單）並確認Brightcove 視頻連接被檢查。這將確保在活動日誌中顯示視頻觀看活動。
  5. 在活動日誌中，查找亮灣活動。展開詳細信息以查看視頻觀看事件的數據。

HubSpot要求將通過其API傳遞的數據分配給已知聯繫人。HubSpot將僅接受已知聯繫人的數據。未知聯繫人的數據將不會同步。如果您不使用潛在客戶表單，則未知用戶將無法轉換為已知用戶，因此不會同步任何數據。
