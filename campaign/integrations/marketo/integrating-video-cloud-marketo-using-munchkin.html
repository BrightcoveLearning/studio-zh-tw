Campaign 模塊允許 Video Cloud 將觀看數據從 Brightcove Players 同步到 Marketo。進入Marketo後，該數據可用於細分，廣告系列邏輯，報告，潛在客戶評分和個性化通信。

有關如何使用 REST API 配置活動模塊以與 Marketo 一起使用的信息，請參閱使用 REST API 將視頻雲與 Marketo 集成 .

與 Marketo 的 Campaign 模塊集成包括兩個主要組件:

  • Brightcove Player插件，用於跟踪視頻觀看數據
  • 從Video Cloud向Marketo發送跟踪數據的同步過程

要求

要將 Campaign 模塊與 Marketo 配合使用，您必須:

  • 擁有啟用了Munchkin API的Marketo帳戶-您將需要您的Munchkin帳戶ID（請參見訪問Marketo Munnchkin以下）;具有Munchkin用戶角色的用戶應該能夠訪問這些值
  • 擁有專業或企業視頻雲帳戶
  • 通過在打開活動模塊時出現的初始啟動屏幕中提交所需信息，為您的帳戶啟用活動模塊
  • 首次使用該模塊時，請單擊服務協議
  • 發布播放器時，請使用高級嵌入（頁內）發布代碼；由於該插件將在iframe中運行，因此標準嵌入（iframe）發布代碼將無法正常工作，因此將無法訪問任何父頁面URL參數，並且可能無法獲得正確的Cookie

特徵

Video Cloud-Marketo集成的一些關鍵功能包括:

  • 潛在客戶表格 -創建Marketo表單，這些表單會在觀眾觀看視頻時自動顯示。有關信息，請參閱為Marketo創建自定義潛在客戶表單。
  • 資料格式 -視頻雲查看數據以頁面瀏覽或虛擬Web訪問的形式交付給Marketo。網頁訪問事件包含指示觀看者參與度的查詢參數。複製到Marketo的數據如下:
    • 影片名稱
    • 影片編號
    • 帳戶ID（視頻雲）
    • 頁面URL（引薦頁面的URL）
    • 播放器ID（視頻雲播放器）
    • 觀看百分比（25％，50％，75％，95％）
  • 數據傳輸 - 當啟用了戰役的 Brightcove 播放器生成查看數據時，查看數據會立即發送到 Marketo，但是，Marketo 平台中的某些報告可能需要一些時間來刷新。Marketo Munchkin 集成不會將查看數據發送到 Campaign。查看數據直接發送到瀏覽器中的Marketo客戶端。
  • 資料保留 -收集的數據可以選擇保留在Video Cloud中長達6個月

訪問Marketo Munnchkin

在配置與Video Cloud的集成時，將提示您輸入Marketo munchkin。請按照以下步驟訪問此值。

  1. 登錄到您的Marketo帳戶。
  2. 點擊行政在頂部導航中。
  3. 在左側導航中，導航至行政 > 一體化 > 蒙奇金。
  4. 複製Munchkin帳戶ID並將其保存在安全的地方。配置 Campaign 模塊時將需要它。

設置連接

請按照以下步驟在Video Cloud和Marketo之間建立連接。

在Video Cloud和Marketo之間建立連接

  1. 打開活動模塊並點擊數據連接在左側導航中。
  2. 點擊添加新連接。
  3. 選擇市場部作為整合。
    筆記: 可用的整合是根據您的 Video Cloud 訂閱而定。
  4. （可選）輸入連接名稱。
  5. 點擊連接。
  6. 選擇高音量（蒙克金）然後點擊保存。
    筆記: 如需有關使用 REST 整合的資訊，請參閱使用 REST API 將視訊雲端與市場整合。
  7. 點擊空的鏈接Marketo帳戶ID部分。粘貼到您的Munchkin帳戶ID然後點擊保存。

現在已建立與Marketo的視頻雲連接並可以使用。

配置集成

建立Video Cloud與Marketo的連接後，即可配置集成。

  1. 默認情況下，連接狀態將已啟用。這連接狀態確定是否將Video Cloud視頻參與度數據發送到Marketo。
    • 已啟用 -收集數據並將其發送到Marketo
    • 殘障人士 -未收集數據（禁用集成）

要刪除與Marketo的集成，請單擊斷開。Campaign 模塊支持與營銷平台的多個連接。點擊添加新連接配置其他集成。

創建支持戰役的播放器

僅當您使用支持廣告系列的視頻播放器時，視頻參與度數據才會發送到 Marketo。有關創建支持戰役的 Brightcove 播放器的信息，請參閱創建支持戰役的播放器 .

傳遞跟踪ID

Campaign 模塊支持將跟踪 ID 值傳遞給 Marketo 的功能。例如，跟踪ID可能用於跟踪廣告系列ID。跟踪 ID 可以附加到顯示已啟用戰役的播放器的 URL 或添加到播放器嵌入代碼。

  • 例子: 附加到URL
  http://www.myurl.com/video.html？ bctrackingid = 12345
  • 例子: 添加到播放器嵌入代碼
  <video data-video-id="6033402539001" data-account="1486906377" data-player="default" data-embed="default" data-application-id class="video-js" 數據 bc-trackingid="12345"控制></視頻><腳本 src="//players.brightcove.net/1486906377/default_default/index.min.js">< /腳本>

然後，Campaign 會將值作為另一個同步字段傳遞給 Marketo。

識別觀眾

具有 Marketo 集成的已啟用戰役的玩家將引用在用戶瀏覽器中創建的 Munchkin cookie (_mkto_trk)。該值在 Campaign 模塊的查看數據中捕獲，用於將查看數據同步到 Marketo。
