Campaign 模塊允許 Video Cloud 將觀看數據從 Brightcove Players 同步到 Marketo。進入Marketo後，該數據可用於細分，廣告系列邏輯，報告，潛在客戶評分和個性化通信。

筆記: 如需有關如何使用 Munchkin ID 配置戰役模組以與 Marketo 搭配使用的資訊，請參閱使用 Munchkin 將視訊雲端與 Marketo 整合。

與 Marketo 的 Campaign 模塊集成包括兩個主要組件:

  • Brightcove Player插件，用於跟踪視頻觀看數據
  • 一個同步過程，用於將跟踪數據從視頻雲發送到Marketo

要求

要將 Campaign 模塊與 Marketo REST 集成一起使用，您必須:

  • 在Marketo中創建一個有權訪問所有API權限的角色
  • 創建啟動點服務
  • 擁有專業或企業視頻雲帳戶
  • 通過在打開活動模塊時出現的初始啟動屏幕中提交所需信息，為您的帳戶啟用活動模塊
  • 首次使用該模塊時，請單擊服務協議
  • 發布播放器時，請使用高級嵌入（頁內）發布代碼；由於該插件將在iframe中運行，因此標準嵌入（iframe）發布代碼將無法正常工作，因此將無法訪問任何父頁面URL參數，並且可能無法獲得正確的Cookie

創建僅API的用戶和LaunchPoint服務

在調用Marketo REST API之前，您需要創建一個具有對所有API的訪問權限的新API角色，創建一個僅API的用戶，然後創建一個唯一標識您的客戶端應用程序的服務。這些任務概述了完成這些任務的步驟。 Marketo文檔。

設置連接

完成Marketo端的配置步驟後，請按照以下步驟設置Video Cloud和Marketo之間的連接。

在Video Cloud和Marketo之間建立連接

  1. 打開活動模塊並點擊數據連接在左側導航中。
  2. 點擊添加新連接。
  3. 選擇市場部作為整合。
    筆記: 可用的整合是根據您的 Video Cloud 訂閱而定。
  4. （可選）輸入連接名稱。
  5. 點擊連接。
  6. 選擇標準（REST API）然後點擊保存。
    筆記: 如需有關使用 Munchkin 整合的資訊，請參閱使用 Munchkin 將視訊雲端與 Marketo 整合。

    您將需要從您的Marketo帳戶中粘貼幾個值。請注意，您必須以Marketo管理員身份登錄才能完成以下步驟。對於以下步驟，請單擊空白鏈接並提供適當的值:

  7. Marketo 帳戶 ID - 要獲得此值，請在 Marketo 中單擊行政 > 一體化 > 芒奇金 .
  8. Marketo REST API 端點 URL - 要獲得此值，請在 Marketo 中單擊行政 > 一體化 > 網頁服務 .
  9. Marketo REST API 身份 URL - 要獲得此值，請在 Marketo 中單擊行政 > 一體化 > 網頁服務 .
  10. Marketo LaunchPoint 客戶端 ID - 要獲得此值，請在 Marketo 中單擊行政 > 一體化 > 發射點然後點擊查看詳情作為此設置的一部分，您之前創建的 LaunchPoint 的鏈接。
  11. Marketo LaunchPoint客戶機密 -要獲得此值，請在Marketo中單擊行政 > LaunchPoint然後點擊查看詳情您之前在此設置中創建的LaunchPoint的鏈接。
  12. 點擊設置。

現在已建立與Marketo的視頻雲連接並可以使用。

筆記: 如果未將正確的權限分配給Marketo用戶，則Studio中將顯示錯誤。請檢查您的權限，然後再試一次。

配置集成

建立Video Cloud與Marketo的連接後，即可配置集成。

  1. 默認情況下，連接狀態將已啟用。這連接狀態確定是否將Video Cloud視頻參與度數據發送到Marketo。
    • 已啟用 -收集數據並將其發送到Marketo
    • 殘障人士 -未收集數據（禁用集成）
  2. 默認，跟踪未知觀看者的數據將設置為殘障人士。啟用此選項將跟踪未知用戶的數據。數據將根據所選的數據保留策略保留。如果數據保留設置為沒有保留，則不會顯示此字段。
  3. 選擇數據保留策略。數據保留策略確定您捕獲的潛在客戶信息數據是否將保留在視頻雲帳戶以及Marketo中以及保留多長時間。將視頻保留在Video Cloud中後，您可以從中斷的同步中恢復。然而，您應該根據您的隱私政策調整保留設置 -不需要在Video Cloud中保留任何用戶數據。數據保留期過後，數據將被刪除。

要刪除與Marketo的集成，請單擊斷開。

創建支持戰役的播放器

僅當您使用支持廣告系列的視頻播放器時，視頻參與度數據才會發送到 Marketo。有關創建支持戰役的 Brightcove 播放器的信息，請參閱創建支持戰役的播放器 .

識別觀眾

具有 Marketo 集成的已啟用戰役的玩家將引用在用戶瀏覽器中創建的 Munchkin cookie (_mkto_trk)。該值在 Campaign 模塊的查看數據中捕獲，用於將查看數據同步到 Marketo。

用戶的電子郵件地址可以通過郵件顯示視頻的頁面上的URL參數。有關更多信息和示例，請參閱活動模塊如何識別查看者 .

驗證整合

配置集成後，請測試集成以確認視頻觀看數據已發送到Marketo。有關驗證集成的信息，請參見Marketo REST集成測試指南。
