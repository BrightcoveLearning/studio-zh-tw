設置新的基於Marketo REST的集成時，測試並確認視頻觀看數據確實已交付給Marketo可能是一個挑戰。在本主題中，您將學習如何系統地測試和確認集成是否有效以及如何將查看的數據和銷售線索發送給Marketo。

有關如何使用 REST API 配置活動模塊以與 Marketo 一起使用的信息，請參閱 使用 REST API 將視頻雲與 Marketo 集成 .

開始之前

在驗證集成之前，假定您具有:

  • 成功完成了Brightcove 活動 REST 集成與 Marketo
  • 至少創建了一個支持戰役的播放器並將其配置為使用 Marketo 連接

配置測試線索表格並發布視頻

出於測試目的，您需要在 Campaign 中創建和配置自定義潛在客戶表單。這是一個 Marketo 表單，您已將其嵌入代碼粘貼到 Campaign 中。即使您不會在生產環境中使用潛在客戶表單，也可以使用該潛在客戶表單來確保您在數據庫和頁面中是已知的。要創建 Marketo 表單並配置 Campaign 以使用它，請執行以下步驟。

  1. 登錄到您的Marketo帳戶。
  2. 創建Marketo表單並設置其樣式。有關創建Marketo表單的信息，請參見為Marketo創建自定義潛在客戶表單。
  3. 右鍵單擊左側導航中的表單，然後單擊嵌入代碼。
  4. 將表單嵌入代碼複製到剪貼板。
  5. 返回活動模塊。
  6. 使用Marketo表單嵌入代碼創建新的潛在顧客表單。
  7. 設置您的自定義窗體以在5秒標記處顯示該窗體。這樣，您將確切知道何時應顯示它。
  8. 選中復選框即使已經知道潛在客戶，也要始終顯示潛在客戶表格。這樣可以確保在測試過程中填寫完表格後繼續顯示該表格。
  9. 保存表格。
  10. 使用Marketo連接配置Brightcove Player，以使用剛創建的Marketo潛在客戶表單。

生成預覽鏈接以觀看視頻

要生成視頻的預覽鏈接，請按照下列步驟操作。

  1. 打開媒體模塊。
  2. 選擇一個視頻並將其發佈到網絡播放器。確保在潛在客戶表單中選擇啟用了戰役的玩家。
  3. 單擊預覽鏈接以在新的瀏覽器窗口中打開視頻。通過這種方法進行測試，可以消除HTML頁面上嵌入的播放器可能引入的任何潛在變量。
  4. 確認視頻已播放，並且線索表顯示在5秒標記處。
  5. 使用唯一的聯繫信息和電子郵件地址（即名人名稱或包含“bctest"的名稱）填寫銷售線索表格，以便以後在Marketo中可以輕鬆找到它。這消除了任何瀏覽器，播放器或數據庫級別的變量。通過以新用戶身份填寫表格，我們可以確保您現在是Marketo的知名用戶。

確認 Campaign 成功捕獲數據

在 Marketo 中查找視頻觀看數據之前，首先確認 Campaign 確實捕獲了視頻觀看。請注意，查看數據最多可能需要 3 小時才能在營銷活動 UI 中顯示。要確認 Campaign 成功捕獲數據，請執行以下步驟。

  1. 打開活動模塊。
  2. 點擊最近所有活動在左側導航欄中。
  3. 確認有一個帶有最近時間戳記的事件，該事件顯示了剛剛觀看的視頻的名稱。事件狀態應為未處理。這表示 Campaign 已捕獲視圖但尚未嘗試將其同步到 Marketo。
  4. 要手動觸發 Campaign 和 Marketo 之間的同步，請單擊同步活動在左側導航中。
  5. 來自選擇連接下拉列表，選擇您的Marketo集成。一種測試同步按鈕應該出現。
  6. 點擊測試同步觸發手動同步。請注意，您限於觸發手動同步的頻率。默認情況下，Campaign 每小時將視圖同步到 Marketo 一次。
  7. 點擊最近所有活動在左側導航欄中。
  8. 您的觀看事件狀態為排隊。數據實際同步到Marketo可能需要15分鐘左右。數據同步後，狀態將更改為以下兩種狀態之一:
    • 已同步 -表示該用戶是已知用戶，並且數據已成功傳送到其活動日誌。幾分鐘後，數據也應該在Marketo中可見。
    • unknown_contact -表示該用戶被視為未知用戶，並且Marketo無法接受數據。如果你得到一個unknown_contact狀態，請檢查您是否使用正確的播放器發布了視頻，是否有Marketo表單以及是否正在使用預覽鏈接（而不是嵌入在頁面中）進行測試。

確認視頻觀看數據在Marketo中

一旦廣告系列顯示已同步狀態，請按照以下步驟驗證數據是否在 Marketo 中。

筆記: Marketo UI 需要 5-10 分鐘才能使用從行銷活動收到的新資料進行更新。
  1. 登錄到您的Marketo帳戶。
  2. 導航到銷售線索數據庫並蒐索新的銷售線索。使用您在填寫潛在客戶表單時使用的唯一名稱或電子郵件地址進行搜索。
  3. 雙擊新的銷售線索以打開銷售線索信息。
  4. 點擊活動日誌標籤。
  5. 使用快速查找...在頁面底部，搜索Brightcove視頻觀看。
  6. 雙擊一行以顯示 Campaign 已填充的字段。彈出窗口將顯示從 Campaign 傳送到 Marketo 的視頻觀看數據，例如播放器 ID、視頻 ID、視頻名稱、觀看百分比和其他字段。
