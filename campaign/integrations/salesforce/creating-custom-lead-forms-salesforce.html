活動模塊提供了創建自定義潛在客戶表單的功能，允許您在 Brightcove 播放器內播放視頻時捕獲觀眾信息。潛在客戶數據將發送到Salesforce並由其進行處理。

要創建自定義銷售線索表單，必須執行以下步驟。

  1. 在Salesforce中創建表單
  2. 複製和編輯表單HTML
  3. 在 Campaign 模塊中創建自定義潛在客戶表單
  4. 將線索表分配給玩家
  5. 在Salesforce中驗證表單數據
筆記: 如需標準潛在客戶表單的資訊，可擷取檢視者資訊並將資料儲存在 Video Cloud 中的資訊，請參閱建立行銷活動商機表單。

在Salesforce中創建表單

Salesforce中的Web-to-Lead功能可用於從網站訪問者生成潛在客戶信息。這些表格也可以顯示在Brightcove播放器內部。下面的步驟將創建一個簡單的表單來捕獲名字，姓氏，電子郵件地址，公司，城市和州/省。有關網絡領先功能的完整詳細信息，請參見Salesforce文檔。

  1. 登錄到您的Salesforce帳戶。
  2. 從設置， 進入Web-to-Lead在快速查找框中，然後單擊網絡對領導。
  3. 點擊創建Web到引線表格。
  4. 選擇要包含在“Web到鉛"表單中的字段。使用添加和刪除箭頭在可用字段列表和選定字段列表，然後使用向上和向下箭頭更改表單上字段的順序。
  5. 進入一個返回網址 .對於此示例，“返回URL"將設置為http://www.brightcove.com。當表單用作 Campaign 模塊內的自定義潛在客戶表單時，將忽略此值。
  6. 點擊產生。將顯示HTML表單。

複製和編輯HTML表單

要使用 Campaign 模塊內的表單，您需要復製表單 HTML。另外，您可能需要對錶單進行一些基本的樣式設置。在下面的步驟中，我們將表單HTML複製到HTML編輯器，然後對錶單進行一些樣式更改。

默認情況下，當窗體顯示在Brightcove播放器內部時，該窗體將顯示在播放器上方的透明背景上。使用CSS設置背景顏色。

  1. 將HTML複製到剪貼板。
  2. 點擊完成的。
  3. 將表單HTML粘貼到您選擇的HTML編輯器中。您可能需要使用編輯器的代碼格式化功能來格式化代碼，以便於閱讀。
  4. 使用您的 HTML 編輯器，添加以下 CSS 以將背景顏色更改為灰色。
    <style>
    body {
    background-color: rgba(204,204,204,0.9);
    margin: 0;
    padding: 20px;
    height: 100%;
    }
    </style>
  5. （可選）要識別哪些潛在客戶是由 Campaign 生成的，可以在<form>標籤:
    <input type="hidden" name="lead_source" value="Video Cloud Campaign">
  6. 將HTML複製到剪貼板。

在 Campaign 模塊中創建自定義潛在客戶表單

在 Salesforce 中生成表單 HTML 後，在營銷活動模塊中創建自定義潛在客戶表單並將潛在客戶表單與啟用營銷活動的播放器相關聯。

  1. 打開活動模塊。
  2. 點擊潛在客戶表格在左側導航欄中。如果已創建任何潛在客戶表單，則將顯示它們。
  3. 點擊創建銷售線索表格。
  4. 給潛在客戶一個描述性的形式名稱。
  5. 選擇一個形式語言。
  6. 選擇表單時間。表單時間決定何時顯示潛在客戶表單。
  7. 將表單類型設置為自定義潛在客戶表格。
  8. 粘貼在表格HTML來自Salesforce。
  9. （可選的）粘貼在確認HTML。確認視圖將顯示5秒鐘，然後視頻將繼續播放。如果未包含確認HTML，則視頻將在提交表單後恢復播放。
  10. （可選的）點擊預覽表格打開表單的預覽。
  11. 點擊保存保存銷售線索表格。

分配線索表格給玩家

一旦創建了線索表，就需要將其分配給玩家。請按照以下步驟將潛在客戶表格分配給玩家。

  1. 點擊競選玩家在左側導航中。
  2. 找到您的播放器並使用潛在客戶表格下拉列表以選擇銷售線索表格。

設置現已完成，您已準備好捕獲潛在客戶信息。當使用支持廣告系列的播放器發布視頻時，潛在客戶表單將根據表單時間設置顯示。

在Salesforce中驗證表單數據

使用自定義潛在客戶表單捕獲的潛在客戶將作為潛在客戶添加到您的Salesforce帳戶中。
