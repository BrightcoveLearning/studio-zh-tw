介紹

根據您配置Cloud Playout頻道的方式，您可以添加各種內容。VOD內容和播放列表可以添加到任何頻道。此外，如果您將頻道配置為支持直播活動，則可以添加直播活動。以下各節提供添加各種內容的步驟。

請注意，您還可以添加保險槓到您的頻道，以及服務器端廣告 , 如果通道是為此配置的。有關如何執行此操作的詳細資訊，請參閱這些主題。

添加視頻點播內容

創建頻道後，或轉到頻道詳細信息時，屏幕將顯示頻道設置:

您可以編輯詳細信息，但是下一步是將內容添加到頻道中。

添加視頻（VOD）步驟

  1. 點擊 新增影片 (如果您在頻道清單頁面上，請按一下頻道名稱以開啟頻道設定頁面，然後按一下 新增視訊 )。
  2. 點擊您要添加到頻道的視頻旁邊的複選框。
  3. 點擊新增影片）
  4. 點擊 儲存。
  5. 該頻道的節目列表將出現:

    請注意，開始時間和結束時間是根據視頻時長自動計算得出的。您可以將視頻拖放到列表中以對其重新排序，並且開始/結束時間將自動調整。

添加播放列表

您也可以將整個播放列表導入您的頻道。請注意以下情況:

  • 您可以同時導入EXPLICIT（手動）列表和智能播放列表。
  • 一種快照當前狀態下的播放列表中的將會被導入（這意味著將來對播放列表進行的更改將不是反映在頻道內容中）。
  • 一次只能導入一個播放列表。
  • 如果導入的播放列表包含一個直播事件，則將創建一個具有相同名稱，描述和持續時間的相同的直播事件。這是因為實況事件始終與通道共享1: 1關係。
  • 播放列表中的視頻將被添加到最後當前頻道節目。
  • 您可以添加VOD內容以及播放列表，並且允許重複。
  • 對總資產限額（100）進行了驗證檢查。如果導入將超過100個資產限制，則導入將失敗。
  • 筆記: 如果頻道的「即時輸入」未啟用，您仍然可以選擇將即時事件排除在匯入之外，匯入包含即時事件的播放清單。
  • 筆記: 如果這是啟用了SSAI的頻道，則不允許直播活動。您仍然可以選擇將即時事件排除在匯入的播放清單中，以匯入包含即時事件的播放清單。
  • 筆記: 僅允許使用具有MP4格式的合格資產。警告指出哪些資產將不符合匯入資格-確認後，只有合格的資產才會被匯入。

添加播放列表步驟

  1. 點擊 匯入播放清單。
  2. 點擊進口您要導入的播放列表。
  3. 如果出現消息指示播放列表中的某些項目無法導入，請單擊繼續如果要導入支持的視頻。
  4. 點擊 儲存。
  5. 如果導入成功，您將在程序列表中看到它:

添加現場活動

注意: 僅在為直播事件啟用了頻道的情況下才適用。

  1. 點擊
  2. 在“添加直播事件"對話框中，輸入名稱，期間， 和描述對於現場活動:

    請注意，現場直播活動的最短持續時間為5分鐘，並且只能以5分鐘為間隔增加或減少持續時間-如果需要，可以在文本框中直接編輯持續時間以調整時間間隔。實時事件的持續時間可以隨時更改，即使當前正在播放。

  3. 添加實時事件後，它們將與VOD項目一起出現在節目列表中:
  4. 您可以像拖動視頻一樣拖動實時事件來重新排序列表。您還可以使用 按鈕來修改實時事件的持續時間。
