Cloud Playout模塊使廣播公司和媒體發行商可以使用VOD視頻和實時流的集合，然後將它們作為線性通道進行流傳輸。該頻道可以流式傳輸到Brightcove Live，Amazon S3或指定的RTMP URL。

要訪問Cloud Playout模塊，請單擊雲播出在導航菜單中。將顯示已創建的所有Cloud Playout頻道的列表（如果有）。

該頁面將顯示以下信息:

  • 頻道名稱 -頻道名稱
  • 開始時間 -頻道的開始時間
  • 時間結束 -頻道的結束時間
  • 循環播放 -指示程序是否將循環
  • 目的地 -頻道的目標位置
  • 創建日期 -頻道創建的日期
  • 最近更新時間 -頻道上次更新的日期
  • 地位 -頻道狀態
    • 草案 -頻道已創建但尚未激活
    • 創作 -頻道正在激活中
    • 預定時間 -頻道已啟用，可以開始播放
    • 開始 -頻道啟動過程開始流式傳輸
    • 跑步 -頻道正在積極流式傳輸
    • 停止 -頻道流媒體正在停止
    • 空閒 -頻道成功運行並已停止
    • 刪除中 -頻道正在刪除
    • 錯誤 -創建或流式傳輸頻道時發生錯誤

建立頻道

要創建新的Cloud Playout頻道，請按照以下步驟操作。

  1. 點擊雲播出在導航標題中。
  2. 點擊 新增:
  3. 進入頻道鑑別細節:
    • 名稱 -頻道名稱。名稱必須包含英文字符，並且允許使用以下特殊字符: _ 。 : / = + - 
    • 描述 -頻道說明
    • 圖片網址 -頻道圖片文件的網址；意味著可以作為通道元數據的一部分檢索的通道圖像
  4. 進入頻道日程細節。
    • 循環播放內容 - 如果上，程序將循環播放，並且結束日期/時間將被啟用。仍可以隨時修改該程序，所做的更改將反映在當前程序和後續循環中。見創建一個循環程序下面的部分。
    • 廣告插入 -如果啟用，則允許在頻道播放期間展示廣告。如果選擇此選項，則不允許實時輸入，並且您只能輸出到Brightcove現場直播或者亞馬遜S3桶。看營利內容。
    • 開始日期/時間 -頻道開始播放的時間。所有資產的開始時間將通過將其持續時間加到該開始時間來計算。僅當通道未處於運行狀態時，才可以更新此時間。
    • 結束日期/時間 -如果未輸入任何值，並且循環播放內容如果未選擇，則播放結束後，流將結束。如果循環播放內容被選中，沒有結束日期/時間，播放流將一直播放到該頻道停止播放為止。此時間可以隨時更新。
  5. 如果要使用實時輸入，請設置現場輸入設為開啟，然後輸入Slate URL，除非您要為平板使用默認的Brightcove MP4。每當停止實時輸入（由於編碼器停止或某些技術問題阻止實時流）時，平板MP4就會播放。注意Slate URL必須是始終可用的靜態URL（不允許使用短暫的CDN URL）。
  6. 進入頻道目的地細節。Cloud Playout同時支持2個輸出。例如，您可以將HLS輸出到S3存儲桶中，並將RTMP提要輸出到Brightcove直播頻道或第3方攝入點。注意合併Brightcove現場直播和實時MP因為不支持同時目的地。
    • 優化顯示 -控制輸出的比特率和尺寸
      • 行動與網路 -如果選中，則為高清輸出（RTMP或HLS單比特率）
      • 智能電視 -如果選中，則為全高清輸出（RTMP或HLS單比特率）
    • 輸出區域 -將在其中創建通道的區域。Brightcove建議選擇地理上最接近大多數觀眾的地區。
    • 輸出到 -選擇通道的輸出方法
      • Brightcove現場直播 -創建或使用以前創建的Brightcove直播頻道。創建新頻道時，該頻道的區域將根據輸出區域被選中。
      • 實時MP -選擇以將流輸出到RTMP或RTP攝取點。RTP不適用於第三方攝入點；您只能在以下情況下使用它廣告插入和Brightcove現場直播已啟用。選中後， RTMP網址字段將出現-輸入實時流式傳輸服務的RTMP輸入URL。
      • 亞馬遜S3桶 -選擇將HLS輸出推送到Amazon S3存儲桶。您將需要輸入存儲桶名稱，區域，訪問密鑰和秘密密鑰。值區必須公開且可寫。
      請注意，輸出到Brightcove 現場活動將自動為您的帳戶生成可計費的實時時間。使用 Cloud Playout 不會產生額外費用，但會收取直播時間的正常費用。輸出到RTMP或Amazon S3不會產生可計費的實時小時數:
  7. 點擊 。將創建狀態為的新頻道草案。

將視頻和實時事件添加到頻道

看向Cloud Playout頻道添加內容有關如何向您的頻道添加VOD內容，播放列表和現場直播的信息。

另請參閱向內容添加保險槓用於在VOD或現場活動項目中添加保險槓。

將視頻和現場活動添加到頻道後，單擊啟用激活頻道。必須先激活頻道，然後才能開始流式傳輸。見使用頻道動作想要查詢更多的信息。

筆記: 若要變更程式中視訊的順序，請按一下視訊並將其拖曳至新位置。

將視頻添加到頻道後，系統創建的播放列表將被創建並在“媒體"模塊中可見。

我們強烈建議您在Cloud Playout模塊中對頻道進行任何更改，並且請勿以任何方式編輯生成的播放列表。

編輯頻道

對於處於DRAFT或SCHEDULED狀態的通道

要編輯草稿或預定的Cloud Playout頻道，請執行以下操作:

  1. 單擊頻道名稱以打開頻道詳細信息。
  2. 點擊
  3. 在對話框中進行適當的編輯，然後單擊

請注意，一旦創建通道，就無法編輯“目標"屬性。

對於運行中的頻道

要編輯正在運行的頻道:

  1. 單擊頻道名稱以打開頻道詳細信息。
  2. 點擊
  3. 點擊
  4. 頻道設置對話框將打開-某些無法更改的選項將不可用。
  5. 進行更改並單擊

創建一個循環程序

要創建循環的程序，請在創建頻道時設置循環播放內容設置為上。將視頻添加到程序後，程序列表將更新，以顯示一系列視頻以及相關的開始/結束時間，以填滿整個頻道的預定時間。當您更改實況事件的持續時間時，已經過去的事件的持續時間將保持不變，但是該事件的以後任何播放都將使用新的持續時間。

刪除頻道

要刪除雲播出頻道，請單擊頻道名稱旁邊的複選框，然後單擊刪除。
