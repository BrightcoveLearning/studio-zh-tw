介紹

將 Cloud Playout 頻道貨幣化的一種方法是通過廣告。投放廣告最可靠的方式是通過服務器端廣告插入或 SSAI。

如果您有興趣將 SSAI 功能添加到您的帳戶，請聯繫您的客戶經理。

本主題涵蓋為 Cloud Playout 頻道啟用 SSAI，以及設置前貼片廣告和後貼片廣告插入。由於插播廣告的工作方式不同，這在將插播廣告添加到頻道 .前貼片廣告、後貼片廣告和插播廣告都可以插入到 VOD 內容頻道中。

對於啟用 SSAI 的頻道，目標必須設置為啟用廣告的 RTP 直播事件:

要求

要將服務器端廣告插入（SSAI）與Cloud Playout結合使用，需要滿足以下條件:

  • 必須為Brightcove Live和SSAI Live啟用該帳戶。
  • Brightcove Live和/或S3必須用作頻道的目的地。

啟用服務器端廣告

創建頻道時，您可以選擇啟用 SSAI，以便使用 Brightcove 的服務器端廣告插入技術顯示服務器端廣告。您還可以啟用

筆記:
  • 使用廣告插入時，只能將頻道輸出到Brightcove直播活動。
  • 如果啟用廣告插入，則可能不會在頻道中使用實時輸入。

配置廣告代碼和填充面板

確保在帳戶管理中配置了廣告代碼和填充選項。有關配置廣告代碼的信息，請參見配置服務器端廣告設置。填充板岩是直播廣告中斷中有空白時可以投放的內容。有關創建填充板的信息，請參見管理實時設置。

筆記: 雲端播放將使用設定為即時設定的預設廣告板。

創建佔位符廣告視頻

您需要指明哪些視頻應用作“佔位符"視頻。“佔位符"視頻將插入到程序列表中，以指示應投放的廣告和廣告的長度。之所以需要這樣做，是因為將服務器端廣告插入與Brightcove Live配合使用時，“佔位符"視頻將被替換為廣告內容。如果您不插入廣告“佔位符"視頻，則實際視頻的一部分將被廣告內容替換。如果廣告持續時間短於返回的廣告，SSAI將顯示廣告板。如果廣告持續時間長於返回的廣告，則將顯示廣告，然後顯示廣告板。

您可能需要創建/上傳視頻源文件，這些視頻源文件的視頻長度應等於您想要的廣告間隔時間（例如2分鐘，4分鐘等）。這些視頻的內容無關緊要，因為它們實際上不會作為程序的一部分進行播放。

要將視頻標記為“佔位符"視頻，請使用“媒體"模塊編輯這些視頻的視頻屬性，並使用標記cp-ad-holder。

為頻道啟用服務器端廣告

創建頻道時，請設置廣告插入選擇上。

筆記: 對於現有的頻道，無法變更「廣告插入」設定。

將廣告佔位符插入程序

您需要將廣告佔位符插入程序列表中應在其中顯示廣告的位置。請按照以下步驟插入廣告佔位符。

  1. 單擊頻道名稱進行編輯。
  2. 在節目列表中，點擊視頻旁邊的複選框後您希望廣告播放哪個。
  3. 點擊插入廣告然後選擇一個廣告時長。廣告時長列表是根據所有廣告中斷的“佔位符"視頻的長度生成的（已標記cp-ad-holder）。
  4. 點擊好的。具有選定長度的“佔位符"視頻將在選定視頻之後插入到程序中。
    筆記: 作為“佔位符"插入的視頻將不會作為程序的一部分播放。此影片僅用於指出要在何處提出廣告請求和廣告長度。
  5. 點擊保存將更改保存到程序中。

選擇廣告配置

程序開始播放後，單擊在Brightcove Live中查看在“實時"模塊中查看頻道。播放器下方的下拉菜單可用於選擇將要使用的廣告配置。

有關服務器端廣告插入的完整信息，請參見Video Cloud SSAI概述話題。
