介紹

為了幫助您進入廣闊的新市場並顯著增加您的電子商務機會，Brightcove 很高興通過與全球電子商務巨頭阿里巴巴的合作，將視頻交付能力引入中國。

由於法規和控制，從國外向中國傳送視頻具有挑戰性。過去，我們提供的解決方案允許 Brightcove 服務通過國內的內容交付網絡分發視頻內容。但是，必要的通信（包括Playback API請求和響應）以及Google Analytics（分析）信標並不可靠，有時甚至無法正常運行。

Brightcove 的中國配送解決方案解決了這些問題，由兩部分組成:

  • 中國境內託管的代理服務器
  • 關聯的中國特定CDN

代理將非視頻流量（播放API請求，Analytics信標和網絡播放器本身）路由到中國的觀眾與來自外部的Brightcove基礎設施之間。

CDN連接到代理並處理視頻和圖像內容。

設置

需要以下設置來支持Brightcove的China Delivery解決方案:

  1. 與您的客戶經理聯繫:

    • 確保您擁有Brightcove Enterprise帳戶
    • 為您的帳戶啟用China Delivery；這是Video Cloud的付費附加組件
    該賬戶可以在中國境內外交付。單獨的帳戶是選擇性的，但不是必需的。

  2. 為中國交付配置播放器。您可以使用Brightcove Web播放器或Native SDK播放器:

    • Brightcove Player的中國配送
    • 使用本地SDK進行中國交付

  3. 取得有效互聯網內容提供商（ICP）許可證

    • ICP許可證上列出的已批准域名或子域名
    • 在批准的域上託管用於發布播放器的網站

局限性

以下是China Delivery的已知限制:

當前限制

  • 播放器配置需要編輯 JSON 和/或播放器管理 API 的使用；通過 Studio 的“一鍵式”播放器配置適用於未來
  • 支持中國交付的 Brightcove Native Android SDK 將於 2021 年第三季度初上市

不支持的功能:

  • 不支持 Brightcove Live Streaming 和 Cloud Playout
  • 不支持動態傳遞規則和靜態 URL

Player 和 SDK 插件可以正常使用，但在中國境內可能無法訪問外部服務，Brightcove 無法為其提供支持。這些包括:

  • 數字版權管理
  • 廣告（CSAI + SSAI）
  • 第三方分析服務
