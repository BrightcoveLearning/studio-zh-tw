360°視頻是同時記錄每個方向的視圖的視頻記錄。使用全向攝像機或攝像機集合拍攝360°視頻。在播放期間，觀看者可以像全景一樣控制觀看方向。

Video Cloud支持上傳和發布360°視頻。下面顯示了一個360°視頻示例。播放視頻後，您可以使用鼠標或箭頭鍵來控制桌面上的視圖，如果在移動設備上觀看，則可以使用陀螺儀。

 

有關瀏覽器支持和360°/ VR視頻插件的已知問題的信息，請參閱360º/ VR視頻插件話題。

錄製360°視頻

通常使用特殊的多台攝像機或包含多台攝像機鏡頭的專用攝像機錄製360°視頻。然後將生成的素材縫合在一起以形成單個視頻。有許多可用的攝像機來錄製360°視頻。本主題中的360°視頻是使用Ricoh Theta S錄製的。Brightcove建議您以高分辨率錄製視頻。

上傳之前

在將360°視頻上傳到Video Cloud之前，視頻文件需要包含某些元數據才能啟用360°播放。空間媒體元數據注入器應用程序可用於將元數據添加到視頻文件中，以指示該文件包含360°視頻。如果360°視頻是在沒有自動添加360°元數據的攝像機上錄製的，請按照以下步驟進行添加:

  1. 下載360°視頻元數據應用程序蘋果電腦或者視窗。
  2. 安裝並打開360°視頻元數據應用程序。
  3. 點擊打開然後選擇視頻文件。
  4. 點擊注入元數據。
  5. 輸入新文件的名稱，然後單擊保存。

現在可以將視頻上傳到Video Cloud。

上載360°影片

拍攝360°視頻以將其投影到球體的內部。這意味著播放器中視頻的可見區域只是整個視頻的一小部分。下面的屏幕快照為您提供了一個大致的概念。紅色矩形內的區域是播放器中可見的區域，但僅佔整個視頻的一小部分。

由於任何時候視頻中只有一小部分會出現在播放器中，因此您希望播放的是高質量的副本，而不會出現質量較低的副本。從本質上講，您是在“放大"部分視頻，而質量差的再現將導致質量差的觀看體驗。上載視頻時，Brightcove建議創建一個具有單個1080p複製和至少一個音頻複製的自定義Dynamic Delivery提取配置文件。

筆記: 有關創建自定義攝取配置文件的信息，請參見管理攝取配置文件。由於移動設備有時具有固定的比特率和分辨率限制，因此應格外小心。部分舊行動裝置可能無法播放高於 1080p 的 360° 影片。

360°影片屬性

所有Video Cloud視頻都具有一組屬性，可以使用Video Cloud Studio或以編程方式使用CMS API進行設置。請注意，並非所有屬性都在Video Cloud Studio中公開。

筆記: 如需所有視訊屬性的清單，請參閱 CMS API 視訊欄位參考文件。

要指示視頻是360°視頻，請使用“媒體"模塊編輯視頻屬性並設置視頻投影財產給360°。

筆記: 這內容管理系統 API也可以用來設置projection財產到等距長方形 .如需詳細資訊，請參閱使用 CMS API 文件設定 360° 視訊屬性。

發布360°視頻

Brightcove提供了幾種發布360°視頻的方式:

  • 網頁上的Brightcove播放器
  • Brightcove社交
  • Brightcove本機播放器SDK
筆記: 若要使用最新版本的 360°/VR 外掛程式，您必須使用 6.13.0 或更新版本的亮灣播放程式。
筆記: 為了讓 360°/VR 外掛程式在 iOS 裝置上正確顯示影片，必須將「播放前預先載入影片」設定設定為「是」作為播放器屬性的一部分。

將360°視頻發佈到網頁

要在Brightcove播放器中播放360°視頻，播放器需要360°/ VR視頻插件。

要使用360°/ VR視頻插件創建新的Brightcove Player，請按照以下步驟操作:

  1. 打開玩家們模塊。
  2. 創建一個新的播放器。
  3. 單擊播放器名稱以打開播放器屬性。
  4. 點擊外掛程式在左側導航欄中。
    筆記: 此外掛程式僅與布萊特灣播放器 6.x 版相容。
  5. 點擊添加插件>自定義插件。
  6. 為了插件名稱， 進入錄像機。
  7. 為了JavaScript網址， 進入: 
    //players.brightcove.net/videojs-vr/1/videojs-vr.min.js
  8. 為了CSS網址， 進入: 
    //players.brightcove.net/videojs-vr/1/videojs-vr.css
  9. 不選項（JSON）是必需的。
  10. 點擊保存然後發布玩家。

現在，您可以使用剛剛創建的播放器發布360°視頻。

使用Brightcove Social發布360°視頻

使用Brightcove Social，可以輕鬆將上傳到您的Video Cloud帳戶的360°視頻發佈到您的社交媒體帳戶。請注意，在使用“社交"模塊同步360°視頻時，用於攝取視頻的攝取配置文件必須配置為保存主文件，否則視頻將無法在360°中同步。當Video Cloud提取360°視頻時，360°元數據不會保留在結果格式中。在沒有母帶的情況下同步360°視頻時，將在同步中使用最高質量的再現。由於格式不包含360°元數據，因此YouTube / Facebook不會知道這是360°視頻。

有關使用Brightcove Social發布視頻的信息，請參閱將視頻發佈到社交媒體平台。

使用Brightcove Native Player SDK發布360°視頻

使用Brightcove Player SDK構建的Android和iOS應用程序可以播放360°視頻。有關使用SDK發布360°視頻的信息，請參見將360°視頻與本機SDK結合使用文檔。
