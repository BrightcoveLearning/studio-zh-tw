腳步

  1. 登錄到Video Cloud Studio。
  2. 在導航標題中，單擊行政然後用戶管理。請注意，ADMIN菜單僅對具有管理員角色的用戶可用。
  3. 輸入電子郵件，名和姓用戶。
  4. 向用戶分配角色:
    • 行政人員 -用戶有權訪問行政菜單
    • 標準 -用戶無權訪問行政菜單
  5. 選擇用戶有權訪問的模塊。關閉模塊會限制用戶可以訪問Studio的哪些部分。
  6. 點擊添加用戶。

用戶將收到一封電子郵件，指出已將其添加到帳戶中。該頁面的正文列出了有權訪問該帳戶的所有用戶。
