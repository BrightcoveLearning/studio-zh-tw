腳步

  1. 打開媒體模塊。
  2. 在側面導航面板中，點擊創建文件夾關聯。
  3. 輸入文件夾的名稱，然後按進入。
  4. 要將視頻添加到文件夾，請選擇一個或多個視頻。
  5. 點擊更多>移至文件夾。
  6. 單擊一個文件夾，然後單擊移動。

所有文件夾都將出現在左側導航欄中。文件夾名稱後面的數字是該文件夾中視頻的數量。

要從文件夾中刪除視頻，請單擊一個文件夾，選擇一個或多個視頻，然後單擊刪除視頻>從文件夾中刪除視頻。
