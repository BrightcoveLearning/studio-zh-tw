先決條件

  • 你有上傳的視頻到您的視頻雲帳戶
  • 您已經創建了一個手動的或者聰明的播放清單

腳步

  1. 打開媒體模塊。
  2. 選擇要發布的播放列表:
    • 如果播放列表出現在側面導航中，請單擊它，然後單擊發布播放列表
    • 點擊所有播放清單在左側導航菜單中，選擇一個播放列表，然後單擊發布和嵌入>發佈到Web
  3. 在裡面選擇播放器下拉列表，選擇要使用的播放器。
  4. 點擊網址在預覽播放器下方，以在新的瀏覽器標籤中將其打開。確認出現播放列表，並且可以播放視頻。
  5. 複製嵌入代碼並將其粘貼到您的網頁中。
