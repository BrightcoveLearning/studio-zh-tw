腳步

  1. 要訪問Brightcove系統狀態頁面，請從Video Cloud Studio內部單擊支持>系統狀態或去status.brightcove.com。

事件狀態將是以下之一:

  • 綠色的 -運作中
  • 藍色的 -通知-系統目前正在運行，但會受到延遲或間歇性停機的影響
  • 黃色的 -部分服務中斷
  • 紅色的 -全面服務中斷
