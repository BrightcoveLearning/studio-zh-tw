要訪問Video Cloud Studio，請導航至: https://studio.brightcove.com。使用您的電子郵件地址和密碼登錄。主頁提供以下功能:

  • 查看系統通知
  • 管理視頻雲設置
  • 選擇一個帳戶並查看用戶個人資料
  • 訪問視頻雲模塊
  • 查看視頻雲儀表板

您最多可以在任何時候進行5次會話。每當您使用相同的電子郵件地址登錄時，都會啟動一個會話。一旦開始第6個會話，最早的會話將失效。Video Cloud Studio會話超時為24小時。

查看系統通知

Video Cloud Studio 標題將顯示一個鈴鐺圖標 當有可用的系統通知時。單擊該圖標將顯示系統通知列表。

Video Cloud Studio 標題將顯示警告圖標 當系統出現問題時。單擊該圖標將打開“Brightcove系統狀態"頁面。

管理設定

您可以通過以下方式訪問Video Cloud Studio設置菜單行政菜單。Video Cloud ADMIN菜單提供對以下內容的訪問:

  • 帳戶信息 -查看有關當前Video Cloud帳戶的信息
  • 管理用戶 -管理有權使用Video Cloud Studio的用戶
  • 帳號設定 -配置常規帳戶設置
  • 上載設定 -提供自定義上傳模塊的選項
  • 攝取個人資料 -管理您的Video Cloud帳戶中的提取配置文件
  • 視頻場 -根據需要標記標準視頻字段，還可以查看，創建和修改帳戶的自定義元數據字段
  • 語言設定 -配置對其他元數據語言的支持
  • 播放權 - 配置播放權限
  • 媒體分享 -將視頻分享到其他帳戶
  • 服務器端廣告設置 - 配置服務器端廣告設置
  • 播放器設定 -提供使用基本用戶名和密碼身份驗證來訪問播放器管理和傳送系統API的選項
  • 整合方式 -將Zoom會議集成到Brightcove Live中
  • 實時設置 -管理實時模塊使用的設置
  • API認證 -創建和管理API憑據
  • 開票 -查看該帳戶的帳單信息
筆記: 您必須是帳戶管理員才能存取管理員功能表。

選擇一個賬戶

如果您有權訪問多個Video Cloud帳戶，則可以使用Video Cloud帳戶選擇器來選擇要訪問的Video Cloud帳戶。要切換帳戶，請點擊您的名稱-帳戶在Video Cloud Studio的右上角。當前帳戶在帳戶名稱之前帶有一個圓點。

點擊我的簡歷鏈接以更新您的用戶信息和密碼。有關更多信息，請參見管理個人資料信息。

訪問模塊

通過Video Cloud Studio菜單可以訪問Studio模塊。通過單擊進入菜單家。

請注意，顯示的模塊基於您的Video Cloud訂閱和用戶配置文件。

  • 上載 -將視頻上傳到您的視頻雲帳戶
  • 媒體 -設置視頻屬性，使用文件夾組織視頻並發布視頻
  • 分析工具 -查看有關視頻和播放器性能的分析
  • 玩家們 -創建播放器並為其設置樣式，並為播放器生成發布代碼
  • 居住 -創建和直播現場直播
  • 雲播出 -為視頻創建實時線性頻道
  • 畫廊 -為您的視頻內容創建視頻門戶
  • 社會的 -將視頻分發到社交媒體帳戶
  • 活動 - 將來自 Video Cloud 的視頻分析連接到營銷自動化平台

查看儀表板

Video Cloud Studio主頁顯示Video Cloud儀表板。信息中心包含一系列“卡片"，可為您帳戶中的視頻提供高級指標。可以使用以下卡:

  • 影片觀看次數 -顯示所選時間段內每天的視頻觀看次數。點擊查看視頻查看詳細信息鏈接將打開效果報告，您可以在其中查看視頻效果。
  • 帳戶匯總 -顯示帳戶中的視頻和播放器總數，以及最近創建和更新的視頻數。單擊一個數字將打開“媒體或播放器"模塊。
  • 設備類型 -顯示所選時間段內用於觀看視頻的設備的類型。點擊按設備類型查看視頻觀看次數鏈接將打開效果報告，您可以在其中按設備類型查看視頻性能。
  • 什麼是新的？ -顯示Video Cloud產品更新。有關更新的完整列表，請參見Video Cloud Studio發行說明。
  • 熱門視頻 -顯示所選時間段內觀看次數最多的視頻。單擊卡片標題鏈接將打開性能報告，您可以在其中查看視頻性能。
  • 查看區域 -顯示選定時間段內的觀看者位置。點擊查看按國家/地區觀看的視頻將打開效果報告，您可以在其中按國家/地區查看視頻觀看次數。
