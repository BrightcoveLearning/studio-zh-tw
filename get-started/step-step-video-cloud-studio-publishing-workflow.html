該主題是視頻雲簡介按需培訓課程。該培訓中的動手練習基於本文檔中的步驟。建議先觀看該培訓，然後再使用本文檔，以便您可以自己完成動手操作步驟。

先決條件

  • Video Cloud帳戶-您可以註冊一個試用帳戶 這裡。
  • 您應該可以使用一些示例視頻，也可以 下載 一些樣本視頻
筆記: 本主題中的步驟和螢幕擷取畫面會使用所提供的範例視訊。

大綱

該主題將針對單個視頻和播放列表進行基本的視頻發布過程。涵蓋以下主題:

  • 上載影片
  • 編輯視頻屬性
  • 創建播放器並為其定型
  • 將視頻發佈到網絡
  • 創建播放列表
  • 發布播放列表

本快速入門未涵蓋視頻雲分析模塊。要了解視頻雲分析，請觀看 視頻雲分析簡介 按需培訓課程。

將視頻上傳到視頻雲

在繼續之前，您應該有一些可以使用的示例視頻，或者下載樣本視頻。

登錄到您的視頻雲帳戶

  1. 在瀏覽器窗口中，導航到https://studio.brightcove.com。
  2. 輸入您的憑據，然後單擊登入。

將本地視頻上傳到Video Cloud

  1. 點擊上載打開上傳模塊。
  2. 在上載模塊中，選擇一個攝取配置文件。
  3. 點擊瀏覽影片並瀏覽並選擇您的視頻或提供的所有示例視頻。
  4. 該頁面將顯示所有視頻和上傳進度。

    上載處於處理狀態時，請勿關閉或離開瀏覽器窗口。這樣做可能會導致上傳不完整。

    上傳完成後，處理完成狀態將會顯示。這表明處理已完成，可以安全地從“上傳"模塊進行導航。

    筆記: 可以單擊編輯文件名旁邊的按鈕可編輯一些視頻元數據。下一個快速入門主題將涵蓋新增視訊中繼資料。

將遠程視頻上傳到Video Cloud

  1. 在裡面上載模塊，單擊輸入網址。
  2. 輸入以下 URL 指向託管在 Amazon S3 賬戶上的文件。
    http://learning-services-media.brightcove.com/vc-quick-start/oystercatcher.mp4
  3. 點擊處理影片開始上傳。

（可選）使用移動設備上傳視頻

  1. 使用移動設備在以下位置打開Video Cloud Studio https://studio.brightcove.com。
  2. 打開上載模塊。
  3. 選擇一個攝取配置文件。
  4. 點擊瀏覽影片。設備上的相機膠卷應用程序應打開。
  5. 選擇要上傳的視頻。上傳將開始，並且將顯示上傳進度。

編輯視頻屬性

在此任務中，您將編輯視頻屬性。

  1. 在Video Cloud Studio中，單擊媒體打開媒體模塊。
  2. 單擊標題為褐鵜鶘視頻，找到影片資訊部分，然後單擊編輯。設置簡短的介紹至Pelican on the rocks。
  3. 添加一個標籤的birds。
  4. 點擊返回影片返回視頻列表。
  5. 單擊標題為http://learning-services-media.brightcove.com/vc-quick-start/oystercatcher.mp4視頻，找到影片資訊部分，然後單擊編輯。設置以下屬性:
    • 名稱: Oystercatcher
    • 簡短的介紹: Oystercatcher on the beach
    • 標籤: birds

    點擊保存完成後，單擊返回影片返回視頻列表。

  6. 編輯兩個山雀和啄木鳥視頻。標記視頻birds。
  7. （可選）編輯一些動物視頻並用標記視頻animals。

創建和样式化播放器

在此任務中，您將創建一個播放器並為其設置樣式。

創建一個新玩家

  1. 在Video Cloud Studio中，單擊玩家們打開玩家們模塊。
  2. 點擊新玩家。
  3. 進入Custom Player輸入名稱，然後單擊保存。

造型播放器

  1. 點擊自訂播放器鏈接以打開播放器屬性。
  2. 確認播放器類型是單個視頻。
  3. 點擊造型風格在左側導航菜單中。
  4. 查看顯示標題和描述。
  5. 選擇一個播放按鈕形狀和播放按鈕位置。
  6. 點擊播放按鈕顏色選擇器，並將顏色設置為紅色陰影。
  7. 確認顏色更改已反映在預覽播放器中。
  8. 點擊發布和嵌入>發布更改發布更改。

將視頻發佈到網絡

在此任務中，您將視頻發佈到網絡上。

發佈單個視頻

  1. 在Video Cloud Studio中，單擊媒體打開媒體模塊。
  2. 點擊旁邊的複選框褐鵜鶘視頻，然後單擊發布和嵌入>發佈到Web。
    筆記: 有關其他發布選項的信息，請參見相應的文檔:
    • 電子郵件登陸頁面 -- 將視頻發佈到電子郵件
    • 社交網絡 -- 將視頻發佈到社交媒體平台
    • 畫廊頁內體驗 -- 將視頻發佈到頁內體驗
  3. 確認“發布"對話框打開。
  4. 在裡面選擇播放器下拉列表選擇自訂播放器。該播放器是在前面的部分中創建的。
    筆記: 在所有視訊雲端帳戶中，預設會建立布萊特灣預設播放器。
  5. 點擊網址在預覽播放器下方，以在新的瀏覽器標籤中將其打開。確認視頻出現並且可以播放。
    筆記: 按一下「縮短」以產生縮短的 URL。
  6. （可選）通過電子郵件將URL發送到您的移動設備。單擊鏈接並確認視頻在您的移動設備上播放。

創建播放列表

在此任務中，您將創建一個手動智能播放列表。

創建手動播放列表

  1. 在Video Cloud Studio中，單擊媒體打開媒體模塊。
  2. 在側面導航面板中，點擊建立播放清單關聯。“新建播放列表"對話框將打開。
  3. 設置播放清單類型至手動的。
  4. 設置播放清單名稱至Favorites。
  5. 查看在側邊欄中顯示播放列表使播放列表出現在邊欄中。
  6. 點擊保存。

將視頻添加到手動播放列表

  1. 點擊所有影片在側面導航面板中顯示該帳戶中的所有視頻。
  2. 點擊您要添加到播放列表中的視頻旁邊的複選框。
  3. 點擊更多>添加到播放列表。
  4. 選擇收藏夾播放列表，然後單擊添加。
  5. 點擊收藏夾側面導航面板中的播放列表，並確認視頻已添加到播放列表中。
  6. 通過單擊一行並將其拖動到列表中的新位置來重新排列視頻的順序。

創建智能播放列表

  1. 在側面導航面板中，點擊建立播放清單關聯。
  2. 設置播放清單類型至聰明的。
  3. 設置播放清單名稱至Birds。
  4. 查看在側邊欄中顯示播放列表使播放列表出現在邊欄中。
  5. 設置標籤到包含一個或多個然後輸入birds標籤。
  6. 點擊保存。
  7. 點擊鳥類側面導航面板中的播放列表，並確認已將小鳥視頻添加到播放列表中。

發布播放列表

在此任務中，您將創建一個新播放器，編輯播放器設置以支持播放列表，然後發布播放列表。

編輯播放器設置以支持播放列表

筆記: 如果選擇反應靈敏調整大小選項後，標準（iframe）代碼效果很好，並且播放器和播放列表都具有響應能力。如果您使用進階 (頁內嵌) 程式碼，播放程式將會回應，但播放清單不會回應。
  1. 在Video Cloud Studio中，單擊玩家們打開玩家們模塊。
  2. 點擊新玩家。命名玩家Playlist Player然後點擊保存。
  3. 找到播放列表播放器並單擊播放器名稱以打開播放器屬性。
  4. 選擇一個播放器類型的播放清單。
  5. 設置長寬比至16: 9。
  6. 設置播放器尺寸到一個寬度的640。
  7. 點擊保存。
  8. 點擊造型風格在左側導航菜單中。
  9. 點擊播放清單類型的播放列表（垂直）。
  10. 點擊回放在左側導航菜單中。
  11. 查看選中時播放視頻。
  12. 查看連續播放模式。
  13. 點擊發布和嵌入>發布更改將更改發布給播放器。

發布播放列表

  1. 點擊媒體打開媒體模塊。
  2. 點擊所有播放清單在側面導航面板中以顯示所有播放列表。
  3. 點擊旁邊的複選框鳥類播放列表。
  4. 點擊發布和嵌入>發佈到Web。
  5. 選擇播放列表播放器。
  6. 點擊網址關聯。播放器將在新的瀏覽器選項卡中打開。確認出現播放列表播放器，並且可以播放播放列表中的視頻。
  7. （可選）通過電子郵件將播放器URL發送到您的移動設備。單擊鏈接並確認播放列表播放器出現，並且可以播放播放列表中的視頻。
