一種

主動媒體
帳戶中的活動視頻數。請注意，這些數據不是實時的-每天從您的視頻庫同步一次。另請注意，如果您正在查看“實時"分析數據（少於3天），則此數據可能不准確。
分析 API
基於REST的應用程序編程接口，使開發人員可以直接以JSON格式檢索視頻雲視頻的分析指標。這分析 API為 Video Cloud Studio 中的 Analytics UI 提供數據。
訪問令牌
臨時的OAuth2令牌，允許訪問REST API，例如CMS API，Analytics API和Dynamic Ingest API。
會員關係
聯合視頻雲播放器的Web發布者。
廣告請求
從視頻雲播放器到廣告服務器的請求（呼叫）。廣告請求是一個HTTP調用，其中包含傳遞信息的鍵/值對，這些信息可幫助廣告服務器針對上下文定位適當的廣告。並非每個廣告請求都會返回一個廣告。
分析 API
這分析 API允許您以編程方式獲取 Video Cloud 帳戶和視頻的分析數據。
蘋果HLS
Apple HTTP Live Streaming，由Apple實施的一種協議，用於按需交付視頻。Apple HLS將視頻分為一系列小文件下載，每個下載都使用HTTP加載一小段視頻。請注意，Apple HTTP Live Streaming與無關直播視頻。
長寬比
視頻幀的寬高比。標準系統為4: 3，HDTV為16: 9。

乙

帶寬
音頻或視頻設備可以編碼或解碼的信號頻率範圍。視頻使用的頻率高於音頻，因此需要更寬的帶寬。
博客
一種基於網絡的出版物，由周期性的貢獻組成，通常按相反的時間順序排列。
Brightcove 播放器
Brightcove Player是一款跨HTML5的高性能，跨平台視頻播放器，可快速加載，在台式機和移動平台上提供高質量的自適應比特率HLS視頻，並與您喜歡的廣告和分析系統集成。
Brightcove Player API
Brightcove Player API可用於以編程方式控制Brightcove Player的外觀和行為。
寬帶視頻頻道
通過Internet分發的相關視頻的集合。
保險槓
一個簡短的視頻剪輯，可以出現在完整長度的視頻之前或播放器加載時。

C

內容管理系統 API
這內容管理系統 API提供對視頻元數據的未緩存讀寫訪問。
編解碼器
用於捕獲數字形式的模擬視頻或音頻的算法。與之比較容器。
容器
視頻容器是一種格式，用於定義如何將視頻的元數據與視頻和音頻數據一起存儲。它沒有定義數據的編碼或壓縮。例如，MP4可以是H.264文件的容器。
內容交付網絡（CDN）
提供聯網服務器的服務，該服務器可以透明地向最終用戶交付內容（尤其是大媒體內容）。兩個著名的CDN是Akamai和Limelight。
內容定位
內容目標是一個過程，用於基於有關該訪問者的已知信息向不同的訪問者顯示不同的內容。
的CSS
級聯樣式表是用於控制HTML頁面設計的標準機制。CSS用於設置Brightcove Player的樣式。
提示點
在視頻中特定位置插入的不可見標記，可用於觸發外部事件，例如插播廣告請求或隱藏式字幕。您可以使用“媒體"模塊設置提示點。

d

DASH（MPEG-DASH）
HTTP上的動態自適應流（也稱為MPEG-DASH）是一種自適應比特率流技術，可通過Internet從常規HTTP Web服務器提供高質量的媒體內容流。如同蘋果的HTTP實時流（HLS）解決方案中，MPEG-DASH通過將內容分成一系列小的基於HTTP的文件段來工作。
域限制
內容安全性的一種基本形式，可防止播放器在指定的域之外運行。
數字版權管理
數字版權管理是一種加密媒體文件並將其打包的方法，以便只能根據策略密鑰對它們進行解密。
動態投放
看多比特率流。
動態攝取 API
動態提取（DI）API用於指示視頻雲檢索視頻資產並為其創建副本。這動態攝取 API也可用於提取縮略圖和海報圖像或 WebVTT 文件作為標題、章節或副標題。

E

編碼
將視頻和音頻數據轉換為可以在播放器中播放的流。
加密
以僅授權方可以觀看的方式對視頻進行編碼的過程。

F

影格速率
每秒顯示視頻剪輯的幀數。

G

地理過濾
通常出於版權擁有的原因，對視頻進行地理限制。

H

H.264格式
也稱為MPEG-4 AVC（高級視頻編碼）。一種用於通過Internet傳遞視頻的文件格式。建議使用H.264格式進行在線視頻傳輸。H.264文件通常使用MP4容器，並具有.mp4文件擴展名。與以前的格式相比，它提供了更大的壓縮率。
HLS
HTTP Live Streaming-由Apple創建的一種視頻格式，可在客戶端流式傳輸的短片段中提供視頻。
的HTML
（超文本標記語言）一種標記語言，用於創建帶有超文本和其他要在Web瀏覽器中顯示的信息的網頁。
HTML5
HTML5是Internet的一種核心技術標記語言，用於為萬維網構造和呈現內容。HTML5是萬維網聯盟（W3C）的HTML標準的第五版。

一世

攝取資料
提取配置文件包含使用以下格式上傳視頻時將使用的轉碼設置: 上載模塊或者動態提取API。
攝取配置文件 API
通過Ingest Profiles API，您可以創建，檢索，更新和刪除Video Cloud帳戶的Ingest配置文件。提取配置文件定義了將視頻添加到Video Cloud時創建的再現。

Ĵ

的JavaScript
一種腳本語言，以其在網站中的使用而聞名，儘管它也用於對包含在其他應用程序中的對象進行腳本訪問。
JPEG（聯合圖像專家組）
一種用於數字圖像的有損壓縮的常用標準方法。JPEG是Brightcove服務中用於縮略圖，視頻靜止圖像和背景圖像的標準圖像格式。

ķ

鍵/值對
播放器發送廣告請求時，它會包含一個或多個鍵/值對（例如category = sports），這些鍵/值對可為廣告服務器提供有關廣告上下文的信息並改善廣告定位。

大號

信箱
一種在寬高比不同的屏幕上顯示寬屏視頻的方法，方法是在圖像的上方和下方留出一個空格（通常為黑條）。
商標
專有圖形，象徵性地代表公司的品牌。
徽標疊加
在視頻播放窗口中顯示的透明小圖像。有時也稱為數字屏幕圖形（DOG），錯誤或水印。
循環/循環
播放列表中的視頻完成後，播放列表將從頭開始播放。
直播視頻
一種協議，通過直接推送捕獲和編碼的流來流式傳輸事件的視頻，這與分發編碼文件以進行流式傳輸相反視頻點播。

中號

媒體庫
視頻雲帳戶中的視頻，播放列表和其他資產的集合。
元數據
元數據是指您在視頻定義中提供的描述，標籤，類型和其他相關信息。
MPAA電影分級系統
由美國電影協會（American Picture Association of America）建立的一種系統，用於根據電影內容對電影進行評分。MPAA分級系統（G，PG，R等）是用於對潛在令人反感的內容進行分類的最公認的系統，但由於MPAA在每個單獨的分級上都有商標，因此通常僅限於電影行業使用。
多比特率流
支持多個版本的功能，或演繹，並可提供最符合檢視器瀏覽器大小和頻寬的轉譯。我們以前將其稱為“動態交付"。
木星與去木星
讀寫各種音頻和視頻流 容器格式。

ñ

本機播放器SDK
Brightcove Native Player SDK提供了構建適用於iOS和Android的具有視頻功能的應用程序所需的工具。

Ø

OAuth
OAuth是授權的開放標準。OAuth 代表資源所有者為客戶端應用程序提供' 對服務器資源的安全委託訪問權限。
OAuth API
OAuth API是RESTful API，允許您獲取授權令牌以訪問Brightcove的其他REST API。

P

回放
在視頻雲播放器中播放視頻媒體。
播放接口
Playback API可以在運行時用於訪問移動應用程序或視頻門戶的視頻和播放列表數據。
播放器管理 API

播放器管理API可用於:

  • 創建一個球員
  • 查看播放器的設置
  • 更改播放器的設置
  • 列出與發布者關聯的所有玩家
  • 發布播放器以將其提供給消費者
播放器發布代碼
用於將Brightcove Player嵌入網頁中的代碼“代碼段"。
播放清單
按特定順序分組在一起以在Video Cloud播放器中播放的視頻的集合。
播放列表 API
播放列表API用於使用JavaScript以編程方式控製播放列表。
政策API
Policy API用於創建或獲取可用於訪問播放API。為Brightcove播放器自動生成策略密鑰。
海報圖片
在視頻播放之前或之後在視頻播放器的播放區域中顯示的靜態圖像。
像素
像素的縮寫，用於CSS。

問

 
 

[R

演繹
使用多比特率流傳輸時，可以創建同一視頻的多個版本，每個版本具有不同的比特率和尺寸。這些版本中的每一個都稱為再現。視頻雲多比特率流媒體根據播放器的尺寸和觀看者的連接速度來選擇最佳再現。
參考編號
發布者分配給視頻或播放列表的唯一ID。參考ID（參考ID）與視頻ID或播放列表ID不同，後者是由Video Cloud生成的唯一ID。參考ID可以在Video Cloud和其他服務（例如發布商的內容管理系統）之間進行協調。
RSS訂閱
真正簡單的聯合組織是一種流行的技術，用於通知用戶網站，博客或Internet電視頻道中內容的更新。

小號

搜索引擎
諸如Google之類的程序，旨在幫助您在網絡上查找信息。
社交網站
一個網站，例如Facebook或Twitter，它為其查看者成員提供了彼此共享消息，媒體和其他信息的能力。
流媒體
一種低位速率編碼格式，旨在通過網絡和Internet使用。流傳輸以編碼後的比特率將小段視頻以匹配用戶的連接速度的方式傳送，因此遠程查看器可以以最小的停頓來播放音頻或視頻，而無需先下載整個視頻文件。

Ť

標籤
用於組織內容的單個關鍵字或短語。Brightcove支持使用標籤，以便發布者對視頻進行組織和分類以及創建播放列表。
縮圖
與播放器，網站和視頻搜索引擎中的視頻列表中使用的視頻相關的小圖像。
基於令牌的身份驗證
基於令牌的身份驗證系統允許用戶輸入其用戶名和密碼，以獲取令牌，從而使他們無需用戶名和密碼即可獲取特定資源。一旦獲得了他們的令牌，用戶就可以向資源提供令牌，該令牌在給定的時間段內提供對特定資源的訪問。
電視家長指南
由美國聯邦通信委員會（FCC）建立的內容分級系統，用於根據內容的年齡適當性對電視節目進行分類。

ü

網址
統一資源定位符（URL）是一種標識資源及其在Internet（例如http://www.brightcove.com）上的位置的方法。

伏特

可變比特率
一種壓縮方案，其中輸入材料的每個單元都可以壓縮為不同的大小。例如，對於MPEG-2視頻，這意味著“更輕鬆"的序列（即沒有運動）可以壓縮到很小的尺寸，而“硬"序列（有很多運動和場景剪輯）可以壓縮到更大的尺寸大小。
視頻

在視頻雲播放器中播放的單個視頻標題。視頻基於視頻文件，該視頻文件可以作為許多受支持的視頻文件格式中的一種開始，但是如果尚未以其中一種格式進行轉換，則可以將其轉碼為H.264文件或FLV文件。另外，視頻可以包含以下元數據:

  • 名稱
  • 簡短的介紹
  • 詳細描述
  • 標籤
  • 視頻靜止圖像
  • 海報圖片
  • 保險槓
  • 徽標疊加
影雲工作室

一個用於訪問和控制視頻雲服務的Internet應用程序。Studio包含用於完成不同任務的一系列模塊:

  • 上載 -將視頻上傳到您的視頻雲帳戶
  • 媒體 -設置視頻屬性，使用文件夾組織視頻並發布視頻
  • 玩家們 -創建播放器並為其設置樣式，並為播放器生成發布代碼
  • 分析工具 -查看有關視頻和播放器性能的分析
  • 居住 -創建和直播現場直播
  • 畫廊 -為您的視頻內容創建視頻門戶
  • 觀眾 -將視頻分析從Video Cloud連接到營銷自動化平台
  • 社會的 -將視頻分發到YouTube，Facebook和Twitter
影片編輯工具
用於在個人計算機上編輯視頻文件的軟件應用程序。
視頻海報圖片
在視頻播放之前或之後在視頻播放器的播放區域中顯示的靜態圖像。
視頻點播
視頻點播。一種協議，允許按需（即，按需）通過流傳輸或漸進式下載來播放分發的視頻文件。與直播視頻。

w ^

WebVTT
Web視頻文本軌道格式。WebVTT文件提供視頻內容的標題或字幕，還提供文本視頻描述，內容導航的章節以及更一般地與音頻或視頻內容時間對齊的任何形式的元數據。Brightcove播放器使用WebVTT格式的字幕。

X

XLSX
Microsoft Excel文件。可以以這種格式導出Analytics數據。
