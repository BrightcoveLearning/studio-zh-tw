可以將文本軌道屬性配置為提供視頻字幕。字幕為希望訪問的個人提供了其他信息或解釋性信息。必須添加字幕，以便聾啞人和聽力不佳的人可以訪問視頻演示文稿的音頻部分。要在Brightcove播放器中顯示視頻的字幕，您需要完成以下任務:

  • 創建字幕文件
  • 配置文本軌道屬性
  • 樣式標題

創建字幕文件

要將字幕添加到視頻，首先需要創建字幕文件。Brightcove播放器支持Web視頻文本軌道格式（WebVTT），SRT，DFXP和SCC格式。WebVTT文件格式（如下所示）是純文本，包含時間文本數據，後跟所需的標題或字幕文本。

筆記

  • SRT，SCC和DFXP字幕將自動轉換為動態交付帳戶的WebVTT。
  • 字幕的文本必須編碼為UTF-8，才能正確顯示為雙字節語言，例如日語和韓語。
  • 由於字幕以HTML顯示，因此其中的任何文本< >方括號將被解釋為標記，並且不會顯示。
WEBVTT 00:00:00.000 --> 00:00:07.080 對齊: 中線: 90% 00:00:07.080 --> 00:00:10.280 align:middle line:84% 大家好，我是 Bob Bailey，Brightcove 的學習專家。00:00:10.280 --> 00:00:12.780 align:middle line:84% 在本視頻中，您將了解 Video Cloud Studio 是如何 00:00:12.780 --> 00:00:14.940 align:middle line:84% 有條理並簡要概述 00:00:14.940 --> 00:00:17.490 align:middle line:90% 你可以使用它來完成。
筆記:
  • 字幕文件名只能包含字母（大寫或小寫），數字，破折號（-）和下劃線（_）。
  • 我們已經看到了由第三方創建的字幕以時間戳開頭的情況:

    01: 00: 00.000

    這將導致字幕被一小時進入視頻。為確保字幕與視頻正確同步，請確保時間戳記以00: 00: 00.000並檢查所有時間戳01小時偏移量。

有關WebVTT格式的更多信息，請參見這個連結。對於您需要支持的每種語言，您應該有一個單獨的字幕文件。儘管有許多第三方工具可以幫助創建字幕文件，但是Brightcove有一些合作夥伴提供轉錄，字幕和翻譯服務。有關Brightcove合作夥伴的完整列表，請訪問夥伴網站的“部分"。

配置文本軌道屬性

創建字幕文件後，它們可以由Brightcove上載並託管，也可以駐留在Internet上可訪問的URL中。

要查看和編輯視頻的文本軌道屬性，請打開“媒體"模塊，單擊視頻標題鏈接，然後找到文字軌道部分。要配置文本軌道屬性，請按照下列步驟操作:

  1. 點擊編輯或雙擊該部分。
  2. 選擇是將文本軌道文件上傳到Video Cloud還是將其託管在外部URL上。
    • 上傳文件 - 點擊瀏覽並從文件系統中選擇文本跟踪文件。該文件將被上傳到Video Cloud。
    • 網址上傳 -輸入文本跟踪文件所在的公共URL，然後單擊上載。該文件將被上傳到Video Cloud。
    • 遠端網址 -輸入文本跟踪文件所在的公共URL。該網址必須少於250個字符。字幕將從遠程URL加載。
      筆記: 如果要在自己的服務器上託管字幕文件，請確保該文件位於Brightcove播放器可以訪問的URL上。承載字幕文件的服務器還應該啟用跨域資源共享（CORS）。你可以在這裡閱讀更多關於 CORS 的信息。
      筆記: 在上載時，帶有括號，問號，引號，空格和其他特殊字符的文本軌道文件名將無提示地失敗。布萊特灣建議使用不含特殊字元的檔案名稱。
      筆記: 您無法在「媒體」模組中編輯共用至帳戶的影片的文字追蹤。
    筆記: [ 預設 (自動顯示)   ] 選項目前對動態傳送標題沒有影響。
  3. 這語字段應包含兩字母語言代碼指定軌道文本數據的語言。
  4. 輸入適當的標籤用於字幕文件。標籤顯示在播放器內部，並允許觀看者選擇一種語言。如果未提供標籤，則顯示語言代碼。
  5. 選擇種類文本軌道。選項包括:
    • 字幕 -與字幕相似，但可能還包括音效和其他音頻信息
    • 字幕 -對話的轉錄或翻譯
    • 內容描述 -旨在作為一個單獨的文本文件，該文件通過屏幕閱讀器描述視頻
    • 章節 -旨在幫助用戶瀏覽視頻
    • 元數據 -有關視頻的信息和內容，默認情況下不打算向觀看者顯示（儘管您可能希望使用JavaScript來顯示）
  6. 輸入MIME類型用於文本跟踪文件。選擇中默認將在播放視頻時顯示字幕（Brightcove Players 6.x及更高版本）。
    請注意，如果您要上傳 SRT 或 DFXP 字幕，您將無法編輯 MIME 類型。
  7. 查看默認（自動顯示）將曲目設置為默認曲目。大多數瀏覽器會嘗試首先顯示默認的文本軌道，但是語言和可訪問性設置可能會覆蓋默認設置。
    筆記: 預設設定不適用於 IE11。
  8. 點擊添加文字軌道保存文本軌道信息。
  9. 點擊添加文字軌道添加另一個文本軌道或保存保存文本軌道。
    筆記: 對文字軌內容所做的變更可能需要 15 分鐘才會出現在已發佈的播放器中。

如果已將任何文本軌道指定為默認軌道，它們將顯示在大膽的。要刪除文本軌道，請單擊刪除圖標 () 旁邊的文本軌道。

請注意，default值不是每個軌道，而是設置為true對於所有相同語言的曲目（srclang )

發布帶有相關字幕的播放器後，抄送按鈕將出現在播放器鑲邊上。點擊抄送按鈕選擇字幕語言並顯示字幕。

編輯字幕屬性

要編輯現有字幕的屬性，請單擊編輯在“文本軌道"面板中，然後單擊要編輯的標題項:

字幕樣式

字幕的默認外觀可能會有所變化，因此您可能希望為字幕設置樣式。有關樣式標題的信息，請參見樣式說明。

局限性

  • 以下設備類型和操作系統支持字幕。
    • iOS 10以上
    • Android 7+
    • Internet Explorer 11+
    • 台式機上最新的Chrome，Safari和Firefox瀏覽器
  • 字幕的長度沒有技術限制，但是為了使整個字幕可見，播放器必須足夠大才能顯示那麼多文字
  • 由於Safari使用本機功能來設置字幕樣式，因此字幕設置菜單中的菜單選項不可用抄送 Safari瀏覽器（iOS和macOS）上的菜單。
  • 如果您在Studio中攝取了無效的WebVTT文件，則該文件將被“處理"，但不會被添加到視頻中。解決方法: 您可以使用驗證您的WebVTT文件這個在線驗證器。
