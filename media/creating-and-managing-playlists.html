播放列表是按照特定順序分組在一起以在播放器中播放的視頻的集合。媒體模塊可輕鬆創建播放列表並將視頻添加到播放列表。創建播放列表後，您可以將播放列表發佈到播放器。有關發布播放列表的更多信息，請參見使用媒體模塊發布視頻和播放列表。

有關播放列表的更多信息，包括最佳做法和有效使用它們的提示，請參閱播放列表簡介。

播放列表類型

播放列表有兩種類型:

  • 手動的 -使用手動播放列表，您可以將特定視頻分配到播放列表，然後手動對其重新排序。
  • 聰明的 -使用智能播放列表，可以根據視頻標籤或其他視頻屬性自動構建播放列表中的視頻列表。智能播放列表是動態的，並會根據媒體庫中的視頻和定義播放列表的規則自動更新。

最多可以將100個視頻添加到播放列表（手動和智能）。可以創建的播放列表數量沒有限制。最初加載到播放器中播放列表中的視頻由播放列表的類型確定:

  • 手動播放列表 -根據手動播放列表中視頻的順序將視頻加載到播放列表中。
  • 智能播放列表 -根據以下內容將視頻加載到播放列表中: 播放順序播放列表設置。

有關以編程方式將其他視頻檢索到播放列表中的信息，請參見播放API分頁播放器樣本。

創建播放列表

要創建播放列表，請打開“媒體"模塊，然後按照以下步驟操作:

  1. 在側面導航面板中，單擊建立播放清單。“新建播放列表"對話框將打開。
    筆記: 您可以通過單擊側面導航按鈕隱藏或顯示側面導航面板 。
  2. 選擇播放列表類型，手動的或者聰明的。
  3. 進入一個播放清單名稱使用以下準則。
    • 名稱不得超過100個字符
    • 非法字符: /
  4. （可選的）。進入一個參考編號和描述。
  5. 查看在側邊欄中顯示播放列表使播放列表出現在邊欄中。
    筆記: 側邊欄中最多可顯示 100 個播放清單。
  6. 點擊保存。

使用智能播放列表

如果要創建智能播放列表，則還可以配置以下設置。

  • 標籤 -標籤用於定義將出現在播放列表中的視頻。這標籤下拉菜單提供以下選項:
    • 包含一個或多個 -包括具有列出的一個或多個標籤的視頻
    • 包含所有 -要求播放列表中的所有視頻都具有列出的所有標籤

    在標籤文本字段中，輸入一個或多個用於定義播放列表的標籤（以逗號分隔）。

  • 播放順序 -控製播放列表中視頻的播放順序。以下選項可用。
    播放順序 描述
    啟用日期（最早的日期） 根據視頻的激活日期，最早的視頻優先
    激活日期（從新到舊） 根據視頻激活的日期，最新的視頻優先
    影片名稱（A-Z） 按視頻名稱的字母順序
    總播放 根據視頻播放總數
    尾隨周播放 根據過去一周的總視頻播放量 [ 1]
    開始日期（最早的日期） 根據視頻的預定開始日期，最早的視頻在前。如果視頻未指定開始日期，則該視頻將不會出現在按開始日期排序的播放列表中。筆記: 視頻計劃僅適用於Video Cloud Enterprise帳戶。
    開始日期（從新到舊） 根據視頻的預定開始日期，首先是最新視頻。如果視頻未指定開始日期，則該視頻將不會出現在按開始日期排序的播放列表中。筆記: 視頻計劃僅適用於Video Cloud Enterprise帳戶。

    筆記

    • [1] 請注意，視頻元數據中維護的連續一周播放數據可能最多保留 2 天，因此如果您在分析模塊中查看過去 7 天的視頻觀看次數，您獲得的視頻可能與您看到的不一致。
  • 影片數量上限 -輸入播放列表中應包含的視頻數量上限。

將視頻添加到手動播放列表

要將視頻添加到手動播放列表，請按照下列步驟操作:

  1. 選中應添加到播放列表中的視頻旁邊的框。
  2. 點擊更多>添加到播放列表在頁面頂部。
  3. 選擇要添加視頻的播放列表。
  4. 點擊添加。添加到手動播放列表的視頻將被添加到播放列表的頂部。

更改播放列表中視頻的順序

要更改播放列表中視頻的順序，請根據要使用的播放列表的類型執行以下步驟。

更改手動播放列表中的視頻順序

  1. 使用以下方法之一打開播放列表:
    • 點擊側面導航面板中的播放列表名稱
    • 點擊所有播放清單在側面導航面板中，找到播放列表，然後單擊播放列表名稱鏈接。
  2. 播放列表中的視頻列表將出現。
  3. 單擊視頻，然後按順序將其拖到新位置。黑線表示新位置。
筆記: 您也可以編輯「播放順序」欄位來變更視訊順序。
通過拖放重新排序確實不是在 IE11 或 Edge 中工作，但您可以通過編輯播放順序場地。

更改智能播放列表中視頻的順序

  1. 如果該播放列表顯示在側邊欄中，請單擊該播放列表，然後單擊該播放列表旁邊的向下箭頭，然後單擊編輯設置。
  2. 如果播放列表未顯示在側邊欄中，請點擊所有播放清單在側面導航欄中找到標題，找到播放列表，然後單擊播放列表名稱鏈接，然後單擊編輯設置。
  3. 選擇一個新的播放順序播放列表。
  4. 點擊保存。

從可用視頻信息加載播放器時，會動態創建智能播放列表，因此順序應始終反映當前狀態。但是，播放器始終通過 Playback API 獲取其信息，該 API 使用緩存最多 20 分鐘的數據。播放列表可能需要很長時間才能刷新，並且可能落後於內容管理系統 API會顯示。

編輯播放列表設置

要編輯手動或智能播放列表的設置，請按照下列步驟操作:

  1. 如果該播放列表顯示在側邊欄中，請單擊該播放列表，然後單擊該播放列表旁邊的向下箭頭，然後單擊編輯設置。
  2. 如果播放列表未顯示在側邊欄中，請點擊所有播放清單在側面導航欄中找到標題，找到播放列表，然後單擊播放列表名稱鏈接，然後單擊編輯設置。
  3. 編輯播放列表設置，然後單擊保存。
請注意，您可能不是將手動播放列表轉換為智能播放列表，反之亦然。

如果該播放列表顯示在側面導航面板中，則可以單擊該播放列表，然後單擊該播放列表旁邊的向下箭頭，然後單擊編輯設置。

刪除播放列表

要刪除播放列表，請按照下列步驟操作:

  1. 點擊所有播放清單顯示所有播放列表的列表。
  2. 點擊播放列表旁邊的複選框以將其刪除。
  3. 單擊刪除按鈕（）。
  4. 點擊刪除播放列表確認刪除。

如果該播放列表顯示在側面導航面板中，則可以單擊該播放列表，然後單擊該播放列表旁邊的向下箭頭，然後單擊刪除。

從手動播放列表中刪除視頻

要從手動播放列表中刪除視頻，請按照下列步驟操作:

  1. 使用以下方法之一打開手動播放列表:
    • 點擊側面導航面板中的播放列表名稱
    • 點擊所有播放清單在側面導航面板中，找到播放列表，然後單擊播放列表名稱鏈接。
  2. 播放列表中的視頻列表將出現。
  3. 點擊要刪除的視頻旁邊的複選框。
  4. 點擊從播放列表中刪除。

您還可以通過編輯視頻屬性從手動播放列表中刪除視頻。在裡面分類部分，從中刪除播放列表手動播放列表部分。
