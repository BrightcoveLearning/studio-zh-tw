要查看和編輯視頻的屬性，請打開“媒體"模塊，然後單擊視頻標題鏈接。視頻屬性頁面將打開。

筆記: 若要同時編輯多個視訊的視訊屬性，請使用「快速編輯」功能。

視頻屬性頁面頂部的按鈕提供了以下功能:

  • 捕獲新的海報和縮略圖圖像
  • 將視頻移到文件夾或者將其添加到播放列表
  • 停用/激活視頻
  • 發布影片

如果視頻被激活，停用按鈕可用於停用視頻。如果視頻被停用，則啟用按鈕可用於激活視頻。被禁用的視頻無法在Brightcove Player中播放。

頁面頂部的預覽播放器可用於觀看視頻。

視頻屬性分為以下幾部分:

  • 影片資訊
  • 分類
  • 影片檔案
  • 媒體分享
  • 音軌
  • 圖片
  • 影片觀看次數
  • 自定義字段
  • 可用性
  • 播放限制
  • 文字軌道

要編輯視頻屬性，請點擊編輯按鈕或雙擊該部分。一些部分允許直接編輯屬性（即分類）。

必填項

您的視頻雲帳戶可能已配置了必填的元數據字段。如果視頻處於非活動狀態，則可以自由編輯和保存其元數據，而無需提供所有必需的元數據。此外，在輸入所有必需的元數據之前，無法激活視頻。如果視頻已經處於活動狀態，則無法在正在編輯的部分中保存任何編輯，除非您在該部分中提供了所有必需的元數據或停用了該視頻。有關使用必填字段的更多信息，請參見設置視頻的必填字段。

影片資訊

視頻信息部分可用於查看和編輯基本視頻屬性。

要編輯屬性，請單擊編輯按鈕或在該部分中雙擊。可以編輯以下屬性:

  • 名稱 -不超過255個字符。
  • ID -系統分配的視頻ID（在所有Video Cloud帳戶中都是唯一的）。
  • 簡短的介紹 -不超過250個字符。
  • 標籤 -一個或多個標籤，以逗號分隔。您可以使用標籤將視頻添加到智能播放列表，或使用CMS API查找視頻。
    • 標籤最多可包含128個字符。如果任何標籤的字符數超過128個，則嘗試保存更改時將顯示錯誤。
    • 標籤不能包含逗號。例如，不支持“屋頂，閣樓+前院"標籤。
    • 一個視頻最多只能包含1200個標籤。
    • 僅將大小寫不同的標籤（小寫，大寫）視為相同的標籤，並且大小寫將自動與原始標籤匹配。
    • 區分大小寫的問題需要在Video Cloud的外部進行處理，例如，使用JavaScript方法修改字符串。
    • 輸入標籤後，按逗號鍵或進入。已添加的標籤將顯示在彩色框中。
  • 參考編號 -發布者分配給視頻或播放列表的唯一ID。限制為不超過150個字符。參考ID與視頻ID或播放列表ID不同，後者是由Video Cloud生成的唯一ID。參考ID可以在Video Cloud和其他服務（例如發布商的內容管理系統）之間進行協調。請注意，您不能批量編輯參考ID，因為兩個視頻不能使用相同的參考ID。參考ID必須始終是唯一的。刪除視頻將允許重新分配先前分配的參考ID。
  • 視頻投影 -指示視頻的錄製方式。
    • 標準 - 默認。定期錄像。
    • 360度 -檢查以表明該視頻是360度視頻。360徽章將出現在媒體模塊的視頻縮略圖中。有關使用360度視頻的更多信息，請參閱處理360°視頻。
  • 啟用離線播放 -選中後，當使用通過設備SDK創建的應用程序時，可以將視頻保存在本地並脫機觀看。僅當該帳戶啟用了脫機播放時，才會顯示此選項。
  • 相關鏈接網址 -以http: //或https: //開頭的標準網址，最多255個字符。
  • 相關鏈接文字 -相關鏈接文字不得超過255個字符。
  • 詳細描述 -不超過5000個字符；不支持嵌入式選項卡和HTML
  • 廣告
    • 自由 -播放視頻時，不會顯示廣告。
    • 廣告支持 -視頻播放時可能會顯示廣告。
      筆記: 如果您的帳戶未啟用廣告，而且您將「廣告」資源設定為「支援廣告」，儲存變更時會收到錯誤訊息。
  • 廣告鍵 -作為廣告服務器網址的一部分傳遞。例如，通過section=blog和anotherKey=value1，您將輸入section=blog&anotherKey=value1。見DFP廣告管理系統幫助有關添加鍵值對的信息。使用廣告鍵時，應將廣告宏添加到廣告服務器網址（配置為廣告宏的一部分）播放器廣告屬性）。廣告宏支持服務器URL中的變量，IMA3插件將用這些變量替換適當的值。有關更多信息，請參見
  • 分享分享 -指示視頻是否已共享到另一個帳戶。有關更多信息，請參見使用媒體模塊進行媒體共享。

點擊保存保存更改。

分類

類別部分顯示手動播放列表視頻已分配給文件夾視頻在播放。要編輯屬性，請直接在此部分中進行更改。按Enter保存更改。

  • 手動播放列表 -該視頻所屬的播放列表。單擊X從播放列表中刪除視頻。有關更多信息，請參見創建和管理播放列表。
  • 文件夾 -視頻分配到的文件夾。看使用文件夾整理視頻想要查詢更多的信息。

按下Enter鍵，更改將被保存。

影片檔案

“視頻文件"部分顯示源文件名，使用的代碼轉換技術以及有關創建的視頻副本的詳細信息。它還提供以下功能:

  • 替換視頻源文件
  • 重新轉碼視頻

在這個例子中動態投放使用了轉碼（在上傳視頻時，使用了“動態投放"或“上下文感知編碼"攝取配置文件）。

筆記: 如果已為帳戶啟用DRM，則DRM切換開關也可能會出現在該部分中。如需 DRM 的詳細資訊，請參閱使用 DRM 擷取內容。

媒體分享

如果視頻已共享到會員帳戶，則將顯示“媒體共享"部分，並列出視頻已共享到的會員。有關媒體共享的更多信息，請參見使用媒體模塊進行媒體共享。

音軌

如果使用“動態傳遞"或“上下文感知編碼"攝取配置文件上載了視頻並對其進行了轉碼，則將顯示“音軌"部分，其中包含有關可用音頻再現的信息。有關在視頻中添加其他音軌的信息，請參見使用媒體模塊將音軌添加到視頻。

筆記: 如果視訊是使用傳統內嵌設定檔轉碼，則不會顯示「音軌」區段。

圖片

圖片部分顯示視頻的海報和縮略圖。海報和縮略圖圖像是通過在中點從源視頻拍攝快照來自動創建的。如果您不喜歡所捕獲的默認圖像，則可以捕捉新圖像或者上傳其他圖片用作海報和縮略圖。將鼠標懸停在海報/縮略圖圖像上，然後單擊查看圖標 () 在新的瀏覽器選項卡中打開圖像。

影片觀看次數

“視頻觀看次數"部分顯示了過去30天的視頻觀看次數。點擊鏈接以打開效果報告或參與度報告，以獲取更多信息。

自定義字段

“自定義字段"部分可用於查看和編輯視頻的自定義字段。要編輯自定義字段值，請點擊編輯按鈕或在該部分中雙擊。字段名稱旁邊的星號（*）表示該字段是必填字段。

筆記: 如需將自訂欄位新增至帳戶的資訊，請參閱建立自訂中繼資料欄位。

可用性

可用性部分顯示視頻可用性設置。有關配置可用性設置的更多信息，請參見配置視頻可用性。

播放限制

“播放限制"部分顯示所有播放限制。有關配置播放限制的更多信息，請參見管理播放權。

文字軌道

“文本軌道"部分可用於將文本軌道添加到視頻。有關配置文本軌道的更多信息，請參見使用媒體模塊向視頻添加字幕。
