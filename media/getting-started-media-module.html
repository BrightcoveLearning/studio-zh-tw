訪問媒體模塊

要訪問媒體模塊，請登錄到Video Cloud Studio然後點擊媒體在導航標題中。

頁面的正文將顯示以下內容:

  • 視頻縮略圖和時長
  • 動態投放指標
    • - 表示視頻已啟用動態交付（視頻是使用 Dynamic Delivery 攝取配置文件攝取的）
    • - 表示視頻未啟用動態交付（視頻是使用舊版攝取配置文件攝取的）
  • 影片名稱
  • 視頻ID-系統分配的視頻ID（在所有視頻雲帳戶中都是唯一的）
  • 更新日期和時間以及用戶ID
  • 創建的日期和時間以及用戶標識
  • 參考編號
  • 地位
    • - 表示視頻處於活動狀態
    • - 表示視頻處於非活動狀態（無法播放）

視頻縮略圖將指示是將視頻共享到帳戶中還是共享給另一個帳戶。在下面的示例中，第一個視頻被共享到該帳戶中，第二個視頻被共享到另一個帳戶中。

視頻縮略圖還指示通過添加360徽章而被標記為360度視頻的視頻。有關使用360度視頻的更多信息，請參閱處理360°視頻。

可以通過單擊一個複選框，然後按住Shift並單擊另一個複選框來選擇多個視頻。通過單擊列標題可以對視頻列表進行排序。媒體模塊使用“粘性"排序，並且會記住排序選擇。當您導航離開並返回或再次登錄時，“媒體"模塊將默認為上次選擇的排序。

創建和更新的日期和時間信息

創建和/或更新視頻的用戶的用戶ID將顯示在“創建日期/更新日期"列中。如果該操作是通過CMS API進行的，由API更新將顯示。請注意，如果將執行操作的用戶從帳戶中刪除，則其ID仍會出現。

在某些情況下，對視頻所做的更改將不會記錄用戶ID，並且可能不會顯示用戶ID:

  • 重新轉換視頻或替換視頻源後，視頻的更新日期將被更新，但是發起操作的用戶將不會被記錄
  • 當圖像（海報/縮略圖）和/或文本軌道被更新時，視頻的更新日期將被更新，但是發起動作的用戶將不會被記錄。
  • 將視頻共享給會員帳戶後，創建日期將不會顯示關聯的用戶ID
  • 使用“社交"或“直播"模塊創建視頻剪輯時，創建的日期將不會顯示關聯的用戶ID

此功能已於2018年12月7日啟用。在此日期之後上傳或編輯的任何視頻都將顯示用戶信息。

訪問側面導航面板

側面導航面板顯示視頻，文件夾和播放列表信息。也可以添加遠程視頻。默認情況下會顯示側面導航面板。

側面導航按鈕 可用於隱藏和顯示側面導航面板。可以通過單擊黑色邊框並拖動來更改側面導航面板的大小。

編輯視頻屬性

要編輯視頻屬性，請單擊視頻名稱以打開視頻屬性頁面。看使用媒體模塊編輯視頻屬性有關使用“媒體"模塊編輯視頻屬性的信息。要同時編輯多個視頻的視頻屬性，請使用快速編輯特徵。

刪除影片

要刪除視頻，請單擊要刪除的視頻的視頻縮略圖旁邊的複選框，然後單擊刪除按鈕 (）。通過單擊確認刪除刪除按鈕。

如果視頻已添加到任何手動播放列表中，則會顯示出來。

要查看視頻所在的手動播放列表，請單擊視頻標題以打開視頻屬性。這分類屬性將顯示已添加視頻的手動播放列表。

將視頻添加到文件夾

可以將視頻添加到文件夾以幫助組織內容。有關使用文件夾的更多信息，請參見使用文件夾整理視頻。

添加遠程視頻

對於遠程視頻，您可以在自己或第三方系統上維護視頻文件，然後向Video Cloud提供視頻文件的URL。有關添加遠程視頻的更多信息，請參見使用遠程資產視頻。

將視頻添加到播放列表

可以創建播放列表以對相關視頻內容進行分組。有關使用播放列表的更多信息，請參見創建和管理播放列表。

發布視頻和播放列表

視頻可以發佈到不同的目的地:

  • 網絡播放器 -提供視頻的網址或生成可以添加到網頁或應用程序中的發布代碼
  • 電子郵件登陸頁面 -提供了將視頻海報圖像嵌入到電子郵件中的功能，並且在單擊時可以在瀏覽器選項卡中打開視頻
  • 社交網絡 -打開一個新的瀏覽器選項卡，可以在該選項卡中使用Brightcove Social發布視頻
  • 畫廊頁內體驗 -使用Brightcove Gallery創建新的頁內體驗

與其他帳戶共享視頻

媒體共享使發布者可以一次上傳視頻，然後將其共享到其他帳戶。這些帳戶可以是您自己的公司帳戶，也可以是其他發布者的帳戶。有關與其他帳戶共享視頻的更多信息，請參見使用媒體模塊進行媒體共享。

導出視頻數據

媒體模塊提供了將所有視頻或視頻子集的視頻元數據導出到.csv文件的功能。有關更多信息，請參見導出視頻數據。

使用快速瀏覽

“快速查看"面板可用於快速查看視頻屬性。要打開和關閉“快速查看"面板，請單擊快速瀏覽。該面板將顯示所選視頻的視頻屬性的子集。

請注意，“快速查看"面板是只讀的。如果選擇了多個視頻，則“快速查看"將顯示最後選擇的視頻的屬性。

搜索視頻

搜索字段可用於搜索您帳戶中的視頻和播放列表。有關更多信息，請參見如何在媒體模塊中搜索視頻和播放列表。

自定義列顯示

可以通過單擊列按鈕 ()，然後選擇應顯示的列。

請注意，列列表還將包括為該帳戶創建的所有自定義字段。隨著選擇更多列，可以水平滾動以查看所有列。

鍵盤快捷鍵

按下？媒體模塊中的快捷鍵將打開可用快捷鍵的列表。
