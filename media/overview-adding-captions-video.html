介紹

可以將隱藏字幕添加到視頻中，以向希望訪問該視頻的個人提供其他信息或解釋性信息。必須添加字幕，以便聾啞人和聽力不佳的人可以訪問視頻演示文稿的音頻部分。

這樣的文本可以用於可訪問性，以提供音軌的翻譯或其他目的，但是所使用的基礎技術是相同的。

字幕文件位置

您可以選擇在何處託管WebVTT字幕文件:

  1. 在遠程URL（遠程資產）上 -在這種情況下，WebVTT文件託管在您的服務器上，並將該文件的URL添加到視頻元數據中。
    筆記: 如果要在自己的服務器上託管字幕文件，請確保該文件位於Brightcove播放器可以訪問的URL上。承載字幕文件的服務器還應該啟用跨域資源共享（CORS）。你可以在這裡閱讀更多關於 CORS 的信息。
  2. 使用Video Cloud上傳（攝取）-在這種情況下，WebVTT文件將被上載（攝取）到Video Cloud。這可以使用Media模塊或API來完成。
筆記: 如果您提取到Video Cloud中的視頻源文件已嵌入608/708字幕，則在處理視頻期間它們將被忽略，並且不是出現在視頻副本中。若要在影片中加入字幕，您必須透過媒體模組、動態內嵌 API 或 CMS API 新增字幕 (適用於遠端代管的字幕)

將字幕與視頻相關聯

有幾種將字幕文件與視頻相關聯的方法:

  1. 使用Studio
  2. 修改播放器發布代碼
  3. 使用API

在Studio中添加字幕

可以使用“媒體"模塊將字幕與視頻相關聯。有關信息，請參閱使用媒體模塊向視頻添加字幕。

修改播放器發布代碼

在這種情況下，<track>標籤已添加到播放器發布代碼中。有關更多信息，請參見以編程方式向視頻添加字幕。

通過API添加字幕

通過API添加字幕的方式會有所不同，具體取決於是提取字幕文件還是將其用作遠程資產。

  • 遠程資產字幕 -如果您希望自己託管字幕（或由託管字幕的提供商提供），則可以使用CMS API將字幕與視頻相關聯。有關信息，請參閱使用CMS API: 添加WebVTT字幕。

  • 向視頻雲添加字幕 -如果您想讓Brightcove託管字幕，則可以使用動態提取API來提取WebVTT文件。有關信息，請參閱攝取WebVTT文件。

筆記: 無論在影片中加入字幕檔案的方法為何，都可以新增超過 1000 個文字軌。

刪除字幕

可以使用“媒體"模塊或CMS API從視頻中刪除字幕。在Studio中，打開“媒體"模塊，然後編輯視頻屬性。編輯文字軌道所需的屬性。

也可以使用內容管理系統 API通過製作一個更新視頻請求並提交一個空的請求正文text_tracks數組（或已從中刪除與該組標題相關聯的對象的數組）。

字幕樣式

可以使用CSS將樣式應用於WebVTT字幕。看樣式說明有關詳細信息。
