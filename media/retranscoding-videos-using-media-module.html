Brightcove保留更新與Video Cloud帳戶關聯的默認提取配置文件的權利。您可能還需要更新自己的自定義攝取配置文件。提取配置文件更改後，您可能希望重新轉換帳戶中的現有視頻，以便它們具有與新的提取配置文件設置相對應的格式。

筆記: 重新轉換現有視頻的功能僅適用於專業帳戶和企業帳戶。如果您對此功能感興趣，請聯絡您的客戶經理。

可以使用“媒體"模塊或使用Dynamic Ingest API對視頻進行重新轉碼。使用動態攝取 API當您有大量需要重新轉碼的視頻時，可能是更好的解決方案。要使用Dynamic Ingest API重新轉碼視頻，請參閱動態提取API概述開發人員文檔。

筆記: 您無法重新轉換已分享到帳戶的影片。
筆記: 只有使用“上傳"模塊或使用Dynamic Ingest API上傳的視頻可以重新轉碼。如果影片無法重新轉碼，重新轉換碼將會失敗。
注意: 只有在將視頻添加到帳戶時已存檔視頻母版，才能重新轉碼視頻。視使用的內嵌描述檔而定，較舊的視訊可能沒有主影片。

重新轉碼視頻的步驟

請按照以下步驟替換視頻的現有副本。

  1. 打開媒體模塊。
  2. 點擊您要重新轉碼的視頻。
  3. 在裡面影片檔案部分，單擊重新轉碼。
  4. 選擇要使用的攝取配置文件。
  5. 查看保留現有的縮略圖和海報圖像保留現有圖像。如果您取消選中此框，則將基於所選攝取配置文件中的圖像設置生成新圖像。
    請注意，如果您選中該框，將保留現有圖像 - 但是，圖片網址會改變 .
  6. 點擊重新轉碼。

在創建新的演繹版本時，視頻將可以使用現有的演繹版本繼續播放。
