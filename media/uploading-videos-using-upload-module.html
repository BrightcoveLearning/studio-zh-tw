上載模塊使用動態提取過程來檢索和轉碼視頻內容。當您選擇要上傳的文件時，Video Cloud會使用基於拉的過程從存儲位置中拉出內容，然後使用Zencoder轉碼服務對內容進行轉碼。

要訪問上載模塊，請單擊上載在導航標題中。

支持的視頻格式

有關視頻格式和編解碼器以及Video Cloud支持哪些源視頻格式的信息，請參閱支持的視頻編解碼器和容器。

選擇攝取配置文件

在上傳視頻之前，請選擇一個攝取配置文件。攝取配置文件包含在對視頻進行轉碼時將使用的轉碼設置。可用攝取配置文件的列表將根據您的視頻雲帳戶的配置方式而有所不同。在下面的屏幕截圖中，兩者上下文感知編碼和動態交付配置文件可用。

筆記: 若要深入了解動態傳遞，請參閱動態傳遞概觀。
筆記: 高效視頻編碼（HEVC）配置文件也可以作為付費附件使用。若要進一步瞭解 HEVC，請參閱 HEVC 影片。

創建的視頻演示文稿基於所選的攝取配置文件。有關Brightcove提供的攝取配置文件的特定詳細信息，請參見用於動態交付和上下文感知編碼的標準攝取配置文件

海報和縮略圖圖像的圖像尺寸基於攝取配置文件中指定的圖像尺寸。

筆記: 如需有關建立其他內嵌設定檔或為帳戶設定預設設定檔的資訊，請參閱管理內嵌設定檔。

上載影片

筆記: 上傳影片之前，請先確定已選取適當的內嵌描述檔。

可以使用以下方法之一上傳視頻:

  • 拖放 -將文件從文件系統拖到“上傳模塊"頁面。導航標題下的整個頁面將成為放置區域。刪除文件後，上傳過程將開始。
  • 文件選擇對話框 - 點擊瀏覽影片打開文件選擇對話框，然後瀏覽並選擇要上傳的文件。
  • 遠端網址 - 點擊輸入網址指定源文件的位置。源文件可以駐留在公共Dropbox文件夾或Amazon S3存儲位置上。點擊處理影片等結束了。
筆記:
  • 沒有文件大小限制。Brightcove已對最大100GB的上傳進行了測試，沒有任何問題。
  • 上傳大文件時，Brightcove建議您設置上傳排隊設置為一次上傳一個文件。
  • 如果在上傳大文件時收到Studio超時錯誤，請嘗試將文件上傳到S3或其他遠程存儲位置，並使用Studio中的URL上傳選項。
  • 無法使用YouTube網址，因為它們不會生成指向視頻文件的直接鏈接。
  • 有關從Google雲端硬盤上傳文件的信息，請參見使用遠程URL上載視頻。
  • 從Dropbox上傳時，如果URL以“？dl = 0"結尾，則在文件上傳之前，您可能需要將其更改為“？dl = 1"。
  • 視頻文件名（包括擴展名）不得超過120個單字節（60個雙字節）字符。如果是的話，視頻將被成功提取，但是您以後將無法對其進行重新轉碼。

選擇文件後，將開始文件上傳。要取消上傳，請單擊取消上傳。點擊解僱從頁面中刪除文件條目。上載處於處理狀態時，請勿關閉或離開瀏覽器窗口。這樣做可能會導致上傳不完整。

上傳完成後，處理完成狀態將會顯示。這表明處理已完成，可以安全地從“上傳"模塊進行導航。

點擊編輯查看攝取配置文件和視頻屬性的子集。如果視頻轉碼尚未開始，則可以更改攝取配置文件。可以編輯以下視頻屬性:

  • 地位 -視頻狀態
    • 積極的 -可以播放視頻
    • 不活躍 -無法播放視頻

    默認情況下，上載視頻時，除非缺少必填字段，否則狀態將設置為“有效"。如果缺少必填字段，則會顯示一條消息，其中必填字段以紅色突出顯示。缺少必填字段的視頻仍可以上傳，但在所有必填字段都具有值之前，它們將保持不活動狀態。有關使用必填字段的更多信息，請參見設置視頻的必填字段。

    筆記: 有一個帳戶級別設置，默認為不活動，可用於將所有上傳的視頻設置為非活動狀態。如需詳細資訊，請參閱管理帳戶設定。
  • 名稱 -視頻標題
  • 簡短的介紹 -視頻的簡短說明
  • 標籤 -與視頻相關的標籤
  • 參考編號 -視頻參考ID
  • 視頻投影 - 選擇360度表示視頻是360度視頻。有關使用360度視頻的更多信息，請參閱處理360°視頻。
  • 啟用離線播放 -選中後，當使用通過設備SDK創建的應用程序時，可以將視頻保存在本地並脫機觀看。僅當該帳戶啟用了脫機播放時，才會顯示此選項。建議僅在使用時設置此選項進階上傳（當用戶必須手動開始上傳處理時）。在視頻上傳過程中設置此設置可能會導致問題。
  • 移至文件夾 -選擇要將視頻分配到的文件夾。有關使用文件夾的更多信息，請參見使用文件夾整理視頻。
  • 自定義字段 -將顯示已標記為必填項的所有自定義字段。

要了解有關視頻屬性的更多信息，請參見使用媒體模塊編輯視頻屬性。

點擊保存保存所有更改並開始轉碼過程。

上載其他資產

當。。。的時候上載其他資產帳戶設置已啟用，視頻屬性面板也會顯示選擇按鈕，可以將海報和縮略圖圖像與視頻一起上傳。點擊選擇選擇海報和縮略圖，然後單擊過程開始轉碼。

請注意，海報和縮略圖圖像的圖像尺寸將基於攝取配置文件中指定的圖像尺寸。有關配置進階上傳帳戶設置，請參見管理帳戶設置。

設置默認文件夾和標籤

上傳視頻時，可以指定要上傳的視頻的文件夾和標籤。文件夾下拉列表將顯示在“媒體"模塊中創建的所有文件夾的列表。點擊+創建文件夾創建新文件夾並將視頻分配到新文件夾。創建新文件夾時:

  • 名稱不得超過100個字符
  • 文件夾名稱在帳戶中必須唯一
  • 非法字符: /

選擇文件夾後，所有上傳的視頻都將添加到該文件夾中。有關使用“媒體"模塊創建視頻並將視頻分配到文件夾的信息，請參閱使用文件夾整理視頻。

標籤也可以在上傳時分配給視頻。輸入以逗號分隔的標籤列表，以分配給視頻。

選擇文件夾或添加標籤後，單擊全部應用保存更改。請注意，該文件夾和標籤將應用於所有已上傳的視頻，正在上傳的視頻或已完成上傳且視頻仍顯示在“上傳"模塊中的位置（解僱按鈕尚未被點擊）。

從移動設備上傳視頻

媒體模塊完全用HTML5編寫，也可以在平板電腦和移動設備上使用。這使您可以使用移動設備拍攝視頻並將其上傳到您的Video Cloud帳戶。要從移動設備上載內容，請打開“上載"模塊，然後選擇一個攝取配置文件。當您單擊瀏覽影片，設備上的照片應用程序將打開。選擇要上傳的視頻，然後單擊完成。來自移動設備的視頻將開始上傳，並且進度將顯示在您的設備上。

常見的上傳錯誤

上傳過程似乎掛起或沒有任何進展

嘗試清除瀏覽器緩存，然後再次上傳視頻。我們已經看到清除瀏覽器緩存將有助於解決視頻上傳問題。

請求時間與當前時間之間的差異太大。

當機器上載視頻的系統時間設置不正確時，將發生此錯誤。這通常在更改時間後發生。確保您的機器設置為正確的時間。
