通常，視頻是使用“上傳"模塊上傳到Video Cloud的。這些文件然後由Brightcove維護。使用遠程資產，您可以在自己或第三方系統上維護視頻文件，然後向Video Cloud提供視頻文件的URL。

筆記: Video Cloud Enterprise發布者可以使用創建遠程資產視頻的功能。如果您有興趣升級您的視頻雲帳戶，請聯繫 Brightcove 以獲取更多信息。

也可以使用Video Cloud CMS API創建遠程資產視頻。有關信息，請參閱開發人員文件。使用創建的遠程資產視頻內容管理系統 API然後可以使用媒體模塊進行管理。

訪問遠程資產

您的所有遠程資產文件必須在公共URL上可用，而對身份驗證或訪問控制沒有任何要求。如果您的遠程資產由CDN託管，則遠程資產的URL應該是CDN中資產的URL，而不是原始服務器上的URL。

創建遠程資產視頻

要創建遠程資產視頻，請按照以下步驟操作。

  1. 打開媒體模塊。
  2. 點擊添加遠程視頻在側面導航面板中。“添加遠程視頻"對話框將打開。
  3. 進入一個影片名稱。視頻名稱將顯示在媒體模塊和播放器中。
  4. 選擇影片類型，居住或者一經請求。
  5. 為了一經請求視頻類型，請輸入期間採用hh: mm: ss格式。
  6. 對於每個格式，輸入渲染網址和格式。點擊添加渲染。
    筆記: 支援通訊協定相對 URL。
  7. 點擊保存保存遠程資產。刷新頁面後，遠程資產視頻將在“媒體"模塊中列出。

遠程資產視頻將沒有海報或縮略圖，但是可以上傳圖像。有關上傳海報和縮略圖的信息，請參見上傳海報和縮略圖圖像。

編輯遠程資產視頻

要編輯遠程資產視頻，請按照以下步驟操作。

  1. 打開媒體模塊。
  2. 單擊遠程資產視頻標題以打開視頻屬性。
  3. 找到視頻文件-遠程資產部分，然後單擊編輯。
  4. 點擊一個來源行以選擇它。單擊源URL將在另一個瀏覽器選項卡中將其打開。
  5. 更新網址和期間如所須。點擊保存保存更改。

局限性

  • 您不能使用圖像捕獲功能為遠程資產視頻創建視頻靜止圖像和縮略圖，但是，可以為遠程資產視頻上傳視頻靜止圖像和縮略圖
  • 遠程資產視頻創建於基本的帳戶創建後無法編輯
  • 遠程資產不支持 SSAI
  • 僅當您可以自定義播放器以與DRM許可證服務器通信時，才支持DRM。Brightcove Player可以通過插件來實現。對於其他播放器，您需要查閱播放器文檔。
  • 分析: Brightcove Analytics系統依賴於系統生成的視頻元數據，例如視頻時長。除非您可以為遠程資產提供此元數據，否則遠程資產的Analytics（分析）數據將不太豐富。
  • 播放保證: 提取到Video Cloud中的所有視頻都可以在Brightcove Player中播放。幾乎所有遠程資產都是如此，但是在某些極端情況下，視頻可能無法播放。例如，我們尚不支持DASH和HLS的一些高級功能。對於非Brightcove播放器，您需要驗證該播放器完全支持視頻格式。
  • 如果計算機的時鐘時間設置在實際時間之前，則觀看者可能無法播放遠程資產視頻
  • 提示點不能添加到實時遠程資產視頻
