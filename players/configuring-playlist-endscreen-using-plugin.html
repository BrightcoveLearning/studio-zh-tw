這播放列表結束畫面插件允許在播放器端屏幕上顯示視頻播放列表。然後，觀看者可以單擊視頻縮略圖以觀看另一個視頻。在此播放器中，已配置播放列表結束畫面。

有關播放器片尾畫面的其他選項的信息，請參閱配置播放器尾屏 .

筆記: 如需以程式設計方式設定播放清單結束畫面的資訊，請參閱開發人員文件。

配置播放列表結束屏幕插件

要使用 Players 模塊實現播放列表結束屏幕插件，請執行以下步驟:

  1. 打開播放器模塊並找到播放器。
  2. 單擊播放器的鏈接以打開播放器屬性。
  3. 點擊外掛程式在左側導航欄中。
  4. 點擊添加插件>Brightcove 插件 .
  5. 為了Brightcove 插件， 選擇播放列表結束畫面 .
  6. 這選項(JSON)包含
    • 播放列表 ID - 要在尾屏上顯示的播放列表 ID
    • 標題（可選）- 顯示在視頻列表上方的標題
    7. {
  "playlistId": "1234567001",
  "title": "Related Videos"
}
  7. 點擊保存然後發布玩家。請注意，更改最多可能需要5分鐘才能顯示在已發布的播放器中。
