播放器模塊提供了為預覽播放器和發布的播放器生成播放器嵌入代碼和播放器URL的功能。通常，當您想在運行時將視頻動態分配給播放器時，可以使用“播放器"模塊為播放器生成嵌入代碼。有關將視頻動態分配給播放器的信息，請參見動態地將視頻分配給播放器。

如果您是視頻雲客戶，希望將播放列表或單個視頻發佈到播放器，請使用“媒體"模塊中的發布功能。有關更多信息，請參見使用媒體模塊發布視頻和播放列表。

預覽播放器與發布的播放器

使用了預覽播放器，因此可以在更改發布之前對其進行預覽。一旦更改被接受，就可以發布播放器，並且更改將反映在任何已發布的播放器中。預覽播放器將顯示Brightcove徽標疊加層，指示不應在生產中使用它。

已發布的播放器已經過優化，並且是將出現在您的網頁上的播放器。該播放器包含對播放器設置所做的所有更改，然後通過單擊發布按鈕。

打開“播放器"模塊，然後選擇一個播放器。點擊嵌入代碼和網址然後點擊預覽播放器或者發行球員。

嵌入代碼的類型

可以生成兩種類型的嵌入代碼:

  • 標準（iframe） -建議在大多數情況下使用。將播放器放置在<iframe>標籤。如果指定了播放器尺寸，則iframe的尺寸將調整為播放器尺寸。如果未指定播放器尺寸，則可以將iframe放置在<div>標籤和播放器的響應方式會調整播放器的大小，以佔據播放器的整個空間<div>。
  • 進階（I 頁數） -允許播放器直接存在於HTML頁面中，而不存在於iframe中。這為開發人員提供了易於訪問播放器以及關聯的屬性和事件的好處。通過使用高級嵌入代碼，您將無法輕鬆地將播放器用作社交媒體應用程序和共享中包含的單個單元。此選項僅適用於已發布的播放器。

為預覽播放器生成嵌入代碼

要為預覽播放器生成嵌入代碼，請執行以下操作:

  1. 打開播放器模塊。
  2. 單擊要為其生成代碼的播放器的行。不要單擊播放器名稱並打開屬性頁面。
  3. 點擊嵌入代碼和網址然後點擊預覽播放器。
  4. 選擇適當的嵌入代碼。

為已發布的播放器生成嵌入代碼

要為已發布的播放器生成嵌入代碼，請執行以下操作:

  1. 打開播放器模塊。
  2. 單擊要為其生成代碼的播放器的行。
  3. 點擊嵌入代碼和網址然後點擊發行球員。
  4. 選擇適當的嵌入代碼。

單擊播放器名稱，然後單擊“複製"，也可以復制已發布的播放器嵌入代碼。發布並嵌入。

將應用程序ID添加到嵌入代碼

過去，發行商通常會創建具有不同名稱的新播放器，以用於網站的不同部分或用於不同的網站。在查看玩家分析時，使用不同的玩家名稱可以更輕鬆地確定哪些玩家獲得的觀看次數最多。通過將應用程序ID參數添加到嵌入代碼中，可以在一個或多個站點中使用同一播放器，並且可以使用該應用程序ID來標識播放器的發布位置。使用播放器應用程序ID可以幫助減少需要創建和維護的播放器數量。有關將應用程序ID添加到嵌入代碼的信息，請參見將應用程序ID添加到播放器嵌入代碼。
