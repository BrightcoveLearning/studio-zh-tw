觀眾

需要使用“播放器"模塊創建和發布視頻的任何視頻發布者。

先決條件

  • 有權訪問播放器模塊的視頻雲帳戶
  • 您的視頻雲帳戶中要發布的視頻；可以下載本快速入門中使用的視頻這裡或者您可以使用自己的內容

創建一個球員

若要創建一個新的播放器，請按照下列步驟。

  1. 登錄到Video Cloud Studio。
  2. 當前的Video Cloud帳戶名稱將顯示在頁面的右上角。如果您有多個帳戶，請點擊當前用戶-當前帳戶鏈接並選擇您要在其中創建播放器的帳戶。
  3. 點擊玩家們導航標題中的鏈接。
  4. 點擊新玩家按鈕。
  5. 進入一個名稱和簡短的介紹為玩家。對於此快速入門，播放器名稱為樣本播放器。
  6. 確認播放器出現在播放器列表中。

使用“播放器"模塊創建新播放器時，將使用Brightcove發布的最新播放器版本來構建播放器。要查看播放器版本，請打開“播放器"模塊，然後單擊要檢查的播放器。玩家信息屬性將顯示已發布的播放器版本。

請注意，所有播放器可能不是同一版本。隨著Brightcove繼續開發和改進Brightcove播放器，將發布新的播放器版本。默認情況下，更新會自動應用到所有播放器。為防止播放器自動更新，您必須設置更新模式設置為手動的（也稱為鎖定播放器）。有關更多信息，請參見設置播放器更新模式和版本。

配置播放器尺寸

通過添加播放器尺寸，寬度和高度參數將添加到iframe嵌入代碼中。如果您將尺寸留空，則可以將播放器放入<div>大小不限的頁面，它將擴展以填充空間。要設置播放器尺寸，請按照以下步驟操作。

  1. 點擊鏈接樣本播放器打開播放器屬性頁面。
  2. 點擊玩家信息在左側導航菜單中。
  3. 設置長寬比至16: 9。
  4. 設置播放器寬度至640。
  5. 點擊保存。

有關“播放器信息"屬性的更多信息，請參見設置播放器信息屬性。

發布和預覽播放器

發布播放器會將更改推送到發布的播放器。有關預覽播放器，已發布的播放器和播放器發布代碼的更多信息，請參見生成播放器嵌入 代碼。

要發布播放器，請按照以下步驟操作。

  1. 點擊發布並嵌入然後發布變更發布播放器。
  2. 確認發布成功。
  3. 點擊播放器網址在新的瀏覽器標籤中打開預覽播放器。播放器將為空，因為尚未將視頻分配給播放器。

動態分配視頻內容給播放器

對於此快速入門，我們將使用URL參數動態地將Video Cloud視頻ID分配給播放器。為此，請按照下列步驟操作。

  1. 打開媒體模塊，然後找到要發布的視頻。視頻ID將顯示在視頻名稱下方。
  2. 將視頻ID複製到剪貼板。
  3. 返回帶有預覽播放器的標籤。要動態傳入要顯示的視頻 ID，請將以下內容附加到 URL 並按 Enter:
    ?videoId=your video id
    筆記: URL 參數名稱區分大小寫。

    完整的URL如下所示:

    https://players.brightcove.net/1486906377/Ctzra7vEL_default/index.html?videoId=5798846643001
  4. 確認視頻出現。

配置樣式屬性

樣式屬性用於自定義播放器的外觀。有關樣式播放器的更多信息，請參見造型球員。

要配置樣式設置，請返回“播放器"模塊，然後按照以下步驟操作。

  1. 點擊鏈接樣本播放器打開播放器屬性頁面。
  2. 點擊造型風格在左側導航菜單中。
  3. 選擇一個播放按鈕形狀和播放按鈕位置。
  4. 在裡面顏色主題部分中，單擊播放按鈕顏色選擇器。
  5. 單擊控件的行駛側上的紅色。單擊正方形以選擇紅色陰影。
  6. 在控件外部單擊以折疊顏色選擇器。
  7. 請注意，在預覽播放器下面有一條消息，指出已對播放器進行了更改。

配置社交共享屬性

啟用社交共享後，共享圖標將出現在播放器底部播放頭旁邊。這將使觀眾可以將您的內容共享給流行的社交媒體網站。有關配置播放器社交屬性的更多信息，請參見配置播放器的社交屬性。

要配置社交設置，請按照下列步驟操作。

  1. 點擊社會的在左側導航菜單中。
  2. 檢查啟用分享複選框。
  3. 選擇所有社交媒體網站。

預覽播放器

頁面上的預覽播放器應反映已對播放器進行的樣式和社交更改。

筆記: 如果返回到我們先前打開的已發布播放器並刷新頁面，您將注意到這些更改未在已發布播放器中進行。顏色和社交變更將不會被推送到已發佈的玩家，直到更改發佈到玩家模組內。

也可以為預覽播放器生成發布代碼。例如，也許營銷部門需要批准對播放器進行的所有更改。您可以向他們發送URL到預覽播放器，以便他們批准更改。要為預覽播放器生成發布代碼，請按照下列步驟操作。

  1. 在“播放器"模塊主頁上，單擊嵌入代碼和網址然後點擊預覽播放器。將顯示“預覽播放器嵌入代碼"對話框。
  2. 點擊播放器網址在新的瀏覽器標籤中打開預覽播放器。
  3. 預覽播放器應佔據瀏覽器頁面的整個寬度，並有Brightcove徽標覆蓋層，指示它是預覽播放器。確認顏色變化和病毒促進變化可見。
  4. 讓預覽播放器保持打開狀態，然後返回“播放器"模塊。

發布最終玩家

在發布更改之前，預覽播放器中的更改將在發布的播放器中不可見。要發布更改，請按照下列步驟。

  1. 在“播放器"模塊中，單擊發布並嵌入然後發布變更發布播放器。
  2. 確認發布成功。
  3. 返回帶有已發布播放器的瀏覽器選項卡。請記住，預覽播放器將顯示Brightcove徽標疊加層。
  4. 刷新頁面，並驗證顏色變化和病毒促進設置現在在已發布的播放器中可見。請注意，更改可能會稍有延遲，更改才會顯示在已發布的播放器中。
