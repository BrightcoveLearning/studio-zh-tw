介紹

社交API可讓您獲取將視頻共享到社交媒體平台的狀態和歷史記錄。通過此快速入門，您可以使用自己的帳戶或Brightcove Learning Services示例帳戶查看可用的請求及其響應。

關於示例應用程序的註釋

  • 這是一個快速入門，旨在為您提供一個用於圍繞Social API構建應用程序的模型。它允許您測試基本請求，但不使用這些請求可以使用的任何參數。要查看這些內容，請查看API參考。
  • 該應用程序還向內容管理系統 API獲取視頻社交共享狀態的初始請求返回的視頻名稱。這僅是為了填充其他請求的視頻選擇器，因為Social API僅返回視頻ID，而不返回其名稱。
提供此示例和相關代碼作為生產開發的指南。該代碼未獲得生產用途的認證，也沒有官方支持。
