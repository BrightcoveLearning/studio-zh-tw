Brightcove Social提供了一些報告，可用於監視發佈到社交媒體帳戶的視頻的效果:

  • 儀錶盤 -概述了社交媒體帳戶上的視頻效果。本主題涵蓋。
  • 社會觀點 -提供共享到社交媒體帳戶的視頻的社交視圖的概述。有關信息，請參閱審查社交觀點分析。
  • 社會參與 -提供與社交媒體帳戶共享的視頻的社交參與（喜歡，分享，轉發等）的概述。有關信息，請參閱回顧社會參與度分析。

開始於選擇時間跨度。要選擇報告，請單擊頁面頂部的標籤。默認情況下顯示儀表板。

儀表板分為以下幾部分:

  • 視頻觀看一目了然
  • 熱門目的地
  • 熱門影片
  • 視頻觀看趨勢

選擇時間跨度

要指定報告的時間段，您可以單擊一個快速日期或使用日曆控件。社交分析數據始終使用GMT時區。

使用快速約會

一系列快速日期位於頁面頂部。提供以下快速日期:

  • 最近7天
  • 最近30天
  • 前一個月
  • 2個月前

單擊快速日期以更新報告。

使用日曆控件

您還可以通過單擊日曆控件來選擇開始和結束日期。

點擊申請更新報告。

視頻觀看一目了然

“視頻觀看一覽"部分將顯示所選時間段內來自視頻雲播放器，Facebook，YouTube的視頻觀看次數和總觀看次數。更新日期是Video Cloud擁有社交分析數據的最新日期。通常，社交平台在更新其API中的視圖和互動數據之前會有24-48小時的延遲。

筆記: Facebook分析中的“使用視頻創建帖子"按鈕將為您的視頻創建一個新的輔助帖子，添加到Brightcove Social創建的視頻中，從而導致出現兩個相同的視頻帖子。布萊特灣社會分析只會呈現來自原始資料的資料。
筆記: Twitter 深入解析 API 不會報告媒體檢視，因此檢視資料無法做為社交分析的一部分。

點擊我的網站和應用程序鏈接將打開“視頻雲分析效果報告"。

請注意，每個社交媒體平台對觀看次數的計數是不同的:

  • 我的網站和應用 -視頻雲在開始播放視頻時對視頻觀看進行計數
  • Facebook -當觀看者觀看視頻至少3秒鐘時，視頻觀看計數
  • YouTube -觀看者觀看視頻“大約" 30秒時計算的視頻觀看次數

熱門目的地

“熱門目的地"部分將顯示在所選時間段內觀看次數最多的前5個目的地。

點擊視頻標題將打開社交觀點分析該視頻。

熱門影片

“熱門視頻"部分將顯示所選時間段內觀看次數最多的前5個視頻。

點擊視頻標題將打開社交觀點分析該視頻。

視頻觀看趨勢

視頻觀看趨勢圖將顯示在選定時間段內發佈到您的社交媒體帳戶的視頻的視頻觀看次數。將鼠標懸停在圖形中的一條線上將顯示該日期的數據。

筆記: Twitter 深入解析 API 不會報告媒體檢視，因此檢視資料無法做為社交分析的一部分。

圖右側的圖例可用於選擇要顯示的詳細程度:

  • 我的網站和應用 -顯示來自Video Cloud播放器和應用的視頻觀看次數
  • Facebook -顯示Facebook上的視頻觀看次數
  • YouTube -顯示YouTube上的視頻觀看次數
  • 全部的 -顯示所有平台上的視頻觀看總數

分析報告中顯示的YouTube觀看次數是從YouTube分析工具概述報告。YouTube不保證此數據為實時數據。因此，您可能會看到幾天的延遲，然後視圖和參與活動才會出現在此報告中。數據進入YouTube Analytics（分析）後，它將由每晚的數據收集者為Brightcove Social Analytics進行提取。

如果您需要更多最新的實時觀看數據，YouTube還可以提供實時報告並估算過去48小時內您最近發布的前25個視頻的觀看數據。您可以找到有關您的YouTube頻道的實時報告這裡。Brightcove社交分析不會收集此實時數據，因為它提供了估算的觀看次數（與實際數據相反），並且僅適用於最近的視頻。

刪除的影片

刪除視頻的社交分析顯示方式取決於視頻的分發方式，如果視頻是自動同步的，則取決於自動同步的設置。

  • 如果視頻是使用社交發布功能分發的，則從視頻雲中刪除視頻不會將視頻從社交平台中刪除，社交將繼續收集視頻的分析數據
  • 如果視頻是使用自動同步功能分發的，則從視頻雲中刪除該視頻可能會從社交平台上刪除該視頻，具體取決於自動同步設置:
    • 如果選取了「刪除 < 社交平台上的分散式影片 >」選' 項，則 Social 模組會從社交平台移除影片，並停止嘗試收集分析資料。
    • 如果選擇了將視頻留在社交平台上的任何選項（這些選項的確切細節因平台而異），則“社交"模塊將繼續嘗試為其收集分析數據。
  • 社交分析將繼續顯示已刪除視頻的任何現有社交分析數據
