Brightcove Social提供了一些報告，可用於監視發佈到社交媒體帳戶的視頻的效果:

  • 儀錶盤 -概述了社交媒體帳戶上的視頻效果。有關信息，請參閱審查社會分析。
  • 社會觀點 -提供共享到社交媒體帳戶的視頻的社交視圖的概述。本主題涵蓋。
  • 社會參與 -提供與社交媒體帳戶共享的視頻的社交參與（喜歡，分享，轉發等）的概述。有關信息，請參閱回顧社會參與度分析。

開始於選擇時間跨度報告。要查看Social Views Analytics報告，請點擊社會觀點頁面頂部的標籤。

社交視圖分析報告分為以下幾部分:

  • 社會觀點
  • 社交見解詳情

社會觀點

“社交視圖"圖將顯示在選定時間段內發佈到您的社交媒體帳戶的視頻的視頻觀看次數。更新日期是Video Cloud擁有社交分析數據的最新日期。通常，社交平台在更新其API中的視圖和互動數據之前會有24-48小時的延遲。

筆記: Twitter 深入解析 API 不會報告媒體檢視，因此檢視資料無法做為社交分析的一部分。

圖右側的圖例可用於選擇要顯示的詳細程度:

  • 我的網站和應用 -顯示來自Video Cloud播放器和應用的視頻觀看次數
  • Facebook -顯示Facebook上的視頻觀看次數
  • YouTube -顯示YouTube上的視頻觀看次數
  • 全部的 -顯示所有平台上的視頻觀看總數

將鼠標懸停在圖形中的一條線上將顯示該日期的數據。

社交見解詳情

“社交視圖詳細信息"部分可用於通過以下方式查看視頻視圖詳細信息:

  • 視圖（默認）
  • 平均觀看時間
  • 平均觀看百分比

做出選擇後，報告將顯示每個視頻的詳細數據。

筆記: Twitter 深入解析 API 不會報告媒體檢視，因此檢視資料無法做為社交分析的一部分。

分析報告中顯示的YouTube觀看次數是從YouTube分析工具概述報告。YouTube不保證此數據為實時數據。因此，您可能會看到幾天的延遲，然後視圖和參與活動才會出現在此報告中。數據進入YouTube Analytics（分析）後，它將由每晚的數據收集者為Brightcove Social Analytics進行提取。

如果您需要更多最新的實時觀看數據，YouTube還可以提供實時報告並估算過去48小時內您最近發布的前25個視頻的觀看數據。您可以找到有關您的YouTube頻道的實時報告這裡。Brightcove社交分析不會收集此實時數據，因為它提供了估算的觀看次數（與實際數據相反），並且僅適用於最近的視頻。

報告底部有控件，可讓您選擇應顯示多少行以及導航到其他頁面。單擊任何視頻標題的鏈接將顯示所選視頻的社交視圖分析。

常見問題

1。當使用社交模塊將視頻發佈到YouTube，然後將該視頻嵌入到另一個站點的YouTube播放器中時，是否仍會收集播放/參與度數據並將其傳遞到社交模塊分析中？

社交分析會收集YouTube報告的所有觀看數據，這些數據會顯示在您的Creator Studio和/或YouTube的分析頁面中。如果YouTube嵌入式播放器的觀看次數對您的分析有所影響，則“社交"模塊將捕獲該情況並進行報告。使用YouTube的嵌入代碼分享>嵌入完全符合給定的功能會導致該嵌入式播放器的觀看次數計入您在YouTube中的分析。但是，編輯嵌入代碼可能會導致YouTube無法收集觀看數據，特別是將嵌入式播放器設置為自動播放視頻似乎會導致YouTube不計算該播放器的觀看次數。

匯出資料

報告數據可以以逗號分隔的值和Excel格式下載。請注意，導出功能直接從分析 API稱呼。導出的數據未按UI數據的方式進行操作，並且當前包含報表中的所有基礎數據，而不僅僅是UI顯示的字段子集。使用API中的原始格式保存所有數據後，您就可以通過UI所提供的方式以外的其他方式來處理數據。
