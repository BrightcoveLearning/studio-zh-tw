Brightcove提供各種支持計劃，以滿足客戶的需求。有關這些程序的信息，請檢查Brightcove支持部分我們的網站。

一個支持選項是支持門戶，可以使用Brightcove支持來打開案件

創建一個新的支持帳戶

您必須先創建一個帳戶，然後才能使用支持門戶網站創建案例。要創建一個新帳戶，請按照下列步驟操作:

  1. 使用以下方法之一訪問支持門戶:
    • 在應用程序標題中，單擊支持>聯繫支持
    • 去https://supportportal.brightcove.com
  2. 點擊報名。
  3. 輸入您的名，姓，電子郵件和密碼。點擊提交。
  4. 確認您已重定向到Brightcove支持門戶主頁。

打開支持案例

要使用支持門戶網站通過Brightcove支持開立案件，請按照以下步驟操作:

  1. 登錄到Brightcove支持門戶，網址為: https://supportportal.brightcove.com。
  2. 點擊提交案例在頁面標題中。
  3. 輸入案例信息:
    • 主題 -案件名稱
    • 描述 -問題，問題或問題的描述
    • 優先事項
      • P1 -嚴重-丟失了Brightcove服務，或嚴重喪失了實時製作播放器的大量發布可用性
      • P2 -重要-Brightcove Service可以運行，但是存在與現場製作播放器特定的預期發布功能有關的阻止問題（即與上載，控制台活動，特定播放器有關）。
      • P3 -正常-未定義為“嚴重"或“重要"的請求（包括帳戶/用戶修改請求）
    • 產品 -與支持請求有關的Brightcove應用程序
    • Brightcove帳戶 -支持請求所涉及的帳戶名
    • 網址 -可以發現問題的頁面網址（如果有）
    • 附加CC -在默認抄送列表的頂部，您可以決定添加協作者的其他電子郵件地址，以告知案件進展情況
  4. 點擊提交。案件詳細信息將顯示。
  5. 點擊寫一個新的評論...鏈接以添加其他評論或上傳屏幕截圖。重現問題的詳細步驟和行為/問題的屏幕快照始終對Brightcove支持有幫助，並可以加快解決案件所需的時間。

查看您的支持案例

注意: 如果您的支持門戶配置文件是行政，您將看到該帳戶的所有案例（其他帳戶用戶創建案例時您不會收到通知）。如果你是一個標準或自行註冊的用戶，您只會看到您打開的案例。< /在旁邊

所有支持案例都可以在支持門戶中輕鬆查看。這案例狀態將是以下之一:

  • 新的 -案例剛剛創建，正在等待Brightcove審核
  • 打開 -案件正在進行中
  • 等待你的回复 -Brightcove支持需要更多信息或已提供解決方案，並且Brightcove正在等待響應
  • 關閉 -案件已解決
  • 封閉等待 -由於發布商未回應，此案已結案；如果發布商聯繫Brightcove，將重新開箱
請注意，已被支持人員刪除的重複案例不會出現在支持門戶中。

要在Brightcove支持下查看您的案例，請按照以下步驟操作:

  1. 登錄到Brightcove支持門戶，網址為: https://supportportal.brightcove.com。
  2. 點擊支持案例在頁面標題中。的列表開箱將顯示。
  3. 點擊病歷鏈接以查看已結案的列表。
  4. 點擊一個案件編號查看案例詳細信息，包括Brightcove支持人員的評論。

單擊列標題將按該列對列表進行排序。

更新支持案例

創建案例後，您可以採取以下三項操作來使案例保持最新狀態:

  • 添加新評論以與我們的支持團隊聯繫
  • 上傳或刪除包含更多信息的附件，以幫助我們盡快發現問題
  • 如果您需要更多關注某個案例，或者它變得比最初預期的更為緊急，請更新案例優先級
  • 更新抄送列表以向案例添加更多觀察者
  • 解決問題後，要求結案

更新默認抄送列表

默認的抄送列表是每個帳戶的電子郵件列表，通知該帳戶的所有案例活動。要更新列表，請按照下列步驟操作:

  1. 轉到右上角，然後單擊用戶圖標。
  2. 點擊我的帳戶。
  3. 輸入新的電子郵件地址或刪除現有的地址帳戶默認抄送列表。
  4. 點擊更新默認抄送列表保存更改。

查看Brightcove系統狀態頁面

Brightcove持續監視所有Brightcove服務的狀態。如果服務中斷，則會在系統狀態頁面上發布註釋。所有預定的維護通知也將被發布。

要訪問“系統狀態"頁面，請登錄支持門戶，然後單擊頁面標題中的“系統狀態"或轉到status.brightcove.com。

要了解有關“系統狀態"頁面的更多信息，請參閱查看Brightcove系統狀態頁面。
